Filter-InfobereichFilter-ArtFormatieren-Filtermodul

Das Formatieren-Filtermodul wird gezeigt, wenn Sie im Filter-Art-Pop-up-Menü im Filter-Infobereich "Formatieren" wählen.

Dieses Modul wird standardmäßig vom Filter "Einfache Formatierung" genutzt.

Dieses Modul bietet die Möglichkeit, die Werte des Dienstes und der Filter über ein Pop-up-Menü "- Variable einsetzen -" mit Ihrem eigenen Text zu kombinieren. Sehr praktisch, um den Ausgabetext mehrerer Filter zu erweitern, irgendeinen Text in Anführungszeichen zu setzen usw.

Dieser Filter gibt den festgelegten Text aus, wobei Variablen durch ihre Werte ersetzt werden. Änderungen können ebenfalls erkannt werden, wenn Sie in der Optionen-Seite die Option "Im Eingabetext nach Änderungen suchen" einschalten.
