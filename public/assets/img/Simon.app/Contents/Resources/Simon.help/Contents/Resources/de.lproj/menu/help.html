Hilfe-Menü

Die Einträge dieses Menüs bringen Sie zu den entsprechenden Seiten in den Websites von Dejal bzw. DSD.net. Wenn Sie einen dieser Menübefehle wählen, wird Ihr Standard-Webbrowser (wie in Ihren Systemeinstellungen eingestellt) aktiviert und die Webseite wird geladen. Sie können das hier simulieren, indem Sie die unten gelisteten Titel anklicken:

Simon-Hilfe:
Nach dem üblichen Hilfe-Suchfeld steht dieser Menüpunkt, der das Handbuch entweder in Apples Hilfeansichtprogramm (Help Viewer) oder in Ihrem Webbrowser anzeigt, abhängig von der nachfolgenden Einstellung.
Hilfe im Help Viewer lesen:
Wählen Sie diese Option, um das Handbuch in Apples üblichem Hilfeansichtprogramm zu lesen. Diese Option ist unter Mac OS X 10.5 (Leopard) nicht verfügbar.
Hilfe im Browser lesen:
Wählen Sie diese Option, wenn Sie dasselbe Handbuch lieber in Ihrem Webbrowser lesen möchten. Der obige Menüpunkt "Simon-Hilfe" sowie alle Hilfe-Buttons im Programm zeigen dann die Handbuchseiten in Ihrem Browser anstatt im Help Viewer.
Simon-Website :
Dieses Kommando öffnet in Ihrem Webbrowser die Produktseite von Simon.
Simon-Änderungen :
Dieses Kommando führt Sie zur Änderungen-Seite von Simon; hier können Sie nachlesen, was sich in der Simon-Entwicklung bisher getan hat. Oben auf der Seite wird gemeldet, ob Ihre Version aktuell oder veraltet ist, so dass Sie diesen Menüeintrag auch als Alternative zum Menüeintrag "Simon > Updates-Einstellungen suchen" nutzen können.
Simon-Extras :
Dieses Kommando öffnet die Seite "Simon-Extras", die aktualisierte und von anderen Benutzern zur Verfügung gestellte Skripte, Bericht-Vorlagen und anderes bereithält.
Häufig gestellte Fragen zu Simon (FAQ) :
Haben Sie eine Frage? Dann ist dies die erste Seite, die Sie aufsuchen sollten! Die Dejal-FAQ-Seiten (Häufig gestellte Fragen und Antworten) beantworten viele immer wieder gestellte Fragen über Dejal, über die Produkte und die Kauf-Möglichkeiten.
Simon-Support-Forum :
Hierüber erreichen Sie das Simon-Forum (auf der Dejal-Website, nur englisch). Dort können Sie neue Funktionswünsche äußern, Fehler melden oder andere Dinge über Simon diskutieren. Wenn Sie sich lieber auf Deutsch über Simon austauschen wollen, können Sie auch unser DSD.net-Mac-Forum besuchen.
MacUpdate-Rezension schreiben :
Sprechen Sie über Simon! Wenn Sie eine Rezension auf MacUpdate schreiben, laden Sie damit andere ein, das Programm ebenfalls einmal auszuprobieren.
iusethis-Rezension schreiben :
iusethis.com ist eine andere großartige Software-Übersicht-Website; auch hier werden Programmempfehlungen immer gern gelesen.
Über Simon twittern :
Mit diesem Menüpunkt können Sie einen Tweet an Twitter vorbereiten, um Ihren Freunden mitzuteilen, dass Sie Simon nutzen (das Absenden erfolgt nicht automatisch).
Simon auf Facebook teilen :
Mit diesem Menüpunkt können Sie eine Nachricht für Facebook vorbereiten, um Ihren Freunden mitzuteilen, dass Sie Simon nutzen (das Absenden erfolgt nicht automatisch).
Dejal bei Twitter: @dejal :
Zeigt die @dejal-Seite auf Twitter, wo immer wieder Neuigkeiten rund um Simon und andere Dejal-Programme (und noch mehr) veröffentlicht werden. Folgen Sie @dejal, um immer aktuell informiert zu sein!
Simon bei Twitter: @DejalSimon :
Zeigt die @DejalSimon-Seite bei Twitter, wo über Simon selbst Nachrichten veröffentlicht werden. Folgen Sie @DejalSimon, um immer aktuell informiert zu sein!
Dejal bei Facebook :
Zeigt die Facebook-Seite von Dejal Systems, wo Neuigkeiten rund um Simon und andere Dejal-Programme (und noch mehr) veröffentlicht werden. Klicken Sie auf der Dejal-Seite auf "Gefällt mir", um immer aktuell von Dejal in Ihrer Neuigkeiten-Seite informiert zu werden!
Website von Dejal Systems :
Dieses Kommando lenkt Ihren Browser auf die Homepage von Dejal Systems. Diese Seite zeigt Produkt-Ankündigungen, Firmen-Nachrichten und Links zu anderen interessanten Bereichen der Dejal-Website. Aktualisieren Sie gegebenenfalls den Inhalt der Seite, um sicherzustellen, dass Sie immer die neuesten Nachrichten sehen.
Dejal-Blog :
Hiermit rufen Sie den Dejal-Blog auf, in dem wir Produkte ankündigen und diskutieren, allgemeine Kommentare zum Mac und vieles mehr äußern.
Dejal-Newsletter :
Abonnieren Sie den Dejal-Newsletter, um (nicht allzu oft) Neues über Simon und Dejal per E-Mail zu erfahren.
Simon-Bestellseite :
Auf dieser Seite können Sie unsere Software kaufen. Sie enthält hilfreiche Informationen und Kauf-Buttons für jedes unserer Produkte.
