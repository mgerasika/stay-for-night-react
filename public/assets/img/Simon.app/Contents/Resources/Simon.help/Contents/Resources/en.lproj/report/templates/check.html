Report TemplatesCheck

This page has information about the Check block, and variables valid within this scope. See the Report Templates page for more information about Simon's report templates.

The Check block includes variables about a check - i.e. values displayed in Simon's Monitor window in the checks table.

Variables are tags in the file that are replaced by another value when the report is used. Variables are enclosed in curly braces, and have a variable name optionally followed by any number of parameters, e.g. {status type="icon"}. See the parameters page for information on the available parameters.

The variables that are available in this scope include:

Variable:Outputs:
DateThe date and time of the check.
DurationThe length of time that the check took to complete.
StatusThe status of the test after this check. Also supports icon output.
