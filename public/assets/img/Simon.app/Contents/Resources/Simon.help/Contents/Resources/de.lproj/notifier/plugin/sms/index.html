Melder-InfobereichMelder-ArtSMS-Melder-Modul

Das SMS-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "SMS" wählen.

Dieses Modul hat zwei Arbeits-Methoden: entweder über den Clickatell -SMS-Gateway-Dienst oder über E-Mail an Ihren Telefon-Anbieter.

In beiden Methoden können Sie mit diesem Modul sehr leicht viele Telefonnummern hinzufügen, um ihnen SMS-Nachrichten zu senden, und Sie können sie sogar aus Ihrem Adressbuch auswählen, falls gewünscht. Clickatell ist ein kostenpflichtiger Dienst; Sie müssen dort zunächst ein Konto anlegen und dann für jede Nachricht bezahlen. Der E-Mail-Modus ist gratis, sofern nicht irgendwelche Gebühren für das Empfangen von SMS-Nachrichten anfallen.

Methode:
Wählen Sie, ob Sie den Clickatell-Dienst oder eine E-Mail an Ihren Telefon-Anbieter nutzen möchten.

Für Clickatell

Clickatell-Konto:
Wenn noch keine Kontodaten eingegeben wurden, landen Sie nach einem Klick auf diesen Button auf einer Seite, die beschreibt, Simon: SMS-Melder-Modul. Sobald die Daten einmal eingegeben wurden, landen Sie nach dem Klick direkt auf der Clickatell-Website.
Benutzername:
Geben Sie hier Ihren Benutzernamen für das Clickatell-Konto ein.
Kunden-ID:
Geben Sie hier Ihre Kunden-ID für das Clickatell-Konto ein, wie sie oben auf der Clickatell-Zentralseite angezeigt wird.
Passwort:
Geben Sie hier Ihr Clickatell-Passwort ein; es wird nicht im Klartext angezeigt.
API-ID:
Clickatell hat eine eigene ID für API-Verbindungen, die Sie hier eingeben müssen.

Für E-Mail

Von:
Für die E-Mail-Methode geben Sie bitte hier Ihre eigene E-Mail-Adresse an; das Modul sendet mit diesem E-Mail-Absender (der gültig sein muss) eine E-Mail an den Telefon-Anbieter.
Betreff:
Der Betroff kann wahlweise mit angegeben werden; er wird am Anfang der SMS-Nachricht eingefügt (und sollte wirklich kurz sein).
Empfänger:
Klicken Sie auf den [+]-Button (Hinzufügen) unterhalb dieser Tabelle, um eine neue Empfänger-Zeile einzufügen, und doppelklicken Sie anschließend auf die Zeile, um sie zu bearbeiten. Sie können einen Namen und eine Telefonnummer durch Komma getrennt eingeben, oder einfach eine Telefonnummer. Für die E-Mail-Methode sollten Sie ebenfalls für jeden Empfänger den Telefon-Anbieter aus dem Pop-up-Menü unterhalb der Tabelle auswählen. Sie können neue Anbieter hinzufügen oder den Domänennamen für existierende Anbieter ebenfalls über dieses Menü ändern.
Adressbuch:
Klicken Sie auf diesen Button unter der Empfänger-Tabelle, um jemanden aus Ihrem Adressbuch hinzuzufügen.
Nachricht:
Geben Sie hier die eigentliche Nachricht für die SMS ein. Sie können hier auch aus dem darunter befindlichen Pop-up-Menü "- Variable einsetzen -" Variablen einfügen. Ebenfalls wird angezeigt, wie viele Zeichen noch übrig sind; die Angabe ist eine Annäherung, da Variablen unterschiedlich lange Texte ergeben können und auch der Betreff und andere Texte noch Platz benötigen. Wenn die Nachricht zu lang ist, teilen die meisten Anbieter sie in mehrere Nachrichten auf, so dass Sie sie nicht unbedingt verkürzen müssen.
