Melder-InfobereichMelder-ArtE-Mail-Melder-Modul

Das E-Mail-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "E-Mail" wählen.

Variable einsetzen:
Variablen sind Marker im Text, die durch andere Werte ersetzt werden, wenn der Dienst benutzt wird. Wählen Sie ein Eingabefeld oder platzieren Sie den Eingabecursor in einem Feld. Anschließend wählen Sie in diesem Menü eine Variable, um sie in dem Feld einzufügen. Um zu erfahren, welche verschiedenen Variablen hier verfügbar sind, sehen Sie bitte im Kapitel Melder-Variablen nach.
Textfeld:
Geben Sie hier den Nachrichtentext für die E-Mail ein, die dieser Melder versenden soll. Sie können Variablen einfügen, die zwischen geschweiften Klammern stehen müssen. Wählen Sie Variablen direkt aus dem Menü Variable einsetzen; Experten können die Variablen direkt eingeben.
An:
Geben Sie hier den E-Mail-Empfänger an. Sie müssen mindestens einen Empfänger entweder im An:- oder im Cc:-Feld angeben, da sonst keine Nachricht verschickt wird. Sie können eine einfache E-Mail-Adresse angeben, wie z.B. "simon@dsd.net" (ohne Anführungszeichen), oder einen Namen und eine E-Mail-Adresse im Standardformat, wie z.B. "Simon bei DSD.net " (ohne Anführungszeichen). Dieses Feld unterstützt Auto-Vervollständigung und bietet ein Menü mit passenden Einträgen aus Ihrem Adressbuch.
Cc:
Dieses Feld entspricht dem An:-Feld, benutzt aber den Carbon-Copy-Header (engl. carbon copy bedeutet soviel wie Durchschlag, Kopie).
Von:
Geben Sie in diesem Feld Ihre E-Mail-Adresse ein. Diese Angabe ist nötig.
Antwort an:
Nutzen Sie dieses Feld, wenn Sie möchten, dass Antworten nicht an die oben angegebene Von:-Adresse geschickt werden sollen.
Betreff:
Geben Sie einen Betreff für die E-Mail-Nachricht vor. Sie können in diesem Feld auch Variablen verwenden: Wenn dieses Feld das aktive Eingabefeld ist (weil es z.B. selektiert ist oder den Eingabecursor zeigt), werden Variablen, die Sie im Pop-up-Menü "Variable einsetzen" auswählen, in dieses Feld eingetragen.
X-Header:
Sie können der E-Mail-Nachricht optional einen Extra-Header mitgeben. Geben Sie den Header-Namen im linken Feld ein und benutzen Sie hierbei keine Leerzeichen. Geben Sie dann den entsprechenden Wert im rechten Feld ein; dieses Feld unterstützt ebenfalls das Einfügen von Variablen. Die obige Abbildung zeigt, wie dieses Feld für den URL des Tests genutzt wird.
Transport:
Lassen Sie den Wert dieses Felds auf Apple Mail, so wird Apple Mail benutzt, um die E-Mail zu versenden. Es wird im Hintergrund gestartet, falls es noch nicht läuft, und die verschickten E-Mails landen anschließend in Ihrem Gesendet-Ordner. Oder wählen Sie einen SMTP-Server oder ein Sendmail-Werkzeug, um damit Nachrichten von diesem Melder versenden zu lassen. Wenn Sie noch keinen SMTP-Server angelegt haben, wählen Sie Server hinzufügen ..., um einen im Dialog E-Mail-Transport-Optionen anzulegen. Der Apple Mail-Transport wird empfohlen, falls Sie nicht mehr Kontrolle benötigen.
Optionen:
Dieser Button zeigt den E-Mail-Transport-Optionen-Dialog, in dem Sie die eingestellte Transportmethode bearbeiten können.
