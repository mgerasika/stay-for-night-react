EinstellungenUpdates-Einstellungen

Das Einstellungen-Fenster wird über den Menüpunkt "Simon > Einstellungen" aufgerufen. In der Seite "Updates" können Sie kontrollieren, ob Simon Ihnen helfen soll, immer auf dem neuesten Stand zu bleiben, indem das Programm automatisch nach Updates sucht. Sie können festlegen, wie oft und nach welcher Art von Updates gesucht werden soll.

Automatisch nach Updates suchen, wenn eine Internet-Verbindung besteht:
Wenn dieser Schalter gesetzt ist, sucht Simon automatisch gemäß der eingestellten Häufigkeit nach neueren Versionen. Die Suche findet nur statt, wenn Simon läuft und wenn Ihr Computer eine Internet-Verbindung hat. Diese Option ist standardmäßig eingeschaltet und es wird empfohlen, diese Option zu nutzen, um das Aktualisieren des Programms so einfach wie möglich zu machen.
Häufigkeit:
Mit diesem Pop-up-Menü legen Sie fest, wie oft Simon nach Updates suchen soll. Die Wahlmöglichkeiten sind: Täglich, Wöchentlich, Monatlich. Nach einer Suche wartet das Programm mindestens den angegebenen Zeitraum, bevor es erneut automatisch nach Updates sucht. Der Standardwert ist Wöchentlich.
Gewünschte Versionen:
Mit diesem Pop-up-Menü können Sie festlegen, ob Sie nur Haupt-Versionen oder auch Beta-Versionen von Simon haben wollen. Der Standardwert ist Nur Haupt-Versionen, falls Sie noch nie eine Beta-Version von Simon genutzt haben, bzw. Beta- und Haupt-Versionen, wenn Sie schon mal eine Beta-Version erhalten haben. You might also want to change this if you are using a localized version of Simon, as beta releases are usually English-only.
Status / Letzte Prüfung:
Diese Zeile zeigt an, ob Simon bei der letzten Suche ein Update gefunden hat oder nicht. Dazu werden Datum und Uhrzeit der letzten Update-Suche gezeigt, entweder für die automatische oder die manuelle Update-Suche.
Jetzt suchen:
Klicken Sie diesen Button, um direkt nach einer neuen Version zu suchen. Dieser Button entspricht dem Menüeintrag "Simon > Updates-Einstellungen suchen".
