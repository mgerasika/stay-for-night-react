Test InfoFilterFormat

The Format filter is displayed when you choose Format in the Filter pop-up menu in the Test Info pane.

The default Simple Format filter is based on the Format filter plug-in.

This filter enables the combination of values from the service and filters, via an insert variable menu, along with your own text. Handy to append the output of multiple filters, wrap something in quote marks, and other uses.

Some filters may display a field to enter the formatting text, others may not display anything, if they are pre-configured to use fixed text.

This filter outputs the specified text, with variables replaced by their values. It can detect changes via the checkbox in the Options page of the Filter Info pane.
