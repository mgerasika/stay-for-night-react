File menu for Services list

The File menu for the Services list contains commands to perform actions on services:

New Service...: This function displays the Service Info pane, described elsewhere.

Duplicate Service...: This function displays the Service Info pane using values from the selected service as a starting point.

Pause Service: This function displays a Pause Service sheet, similar to the Pause Test one. This is useful if you want to avoid certain kinds of service, without having to pause all tests of that kind.
