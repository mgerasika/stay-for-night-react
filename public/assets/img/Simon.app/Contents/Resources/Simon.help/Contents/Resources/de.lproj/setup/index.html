Installation und Einstellungen

Simon wird als ZIP-Archiv geliefert. Um das Programm aus dem Archiv zu entpacken, machen Sie einfach nach dem vollständigen Download einen Doppelklick auf das Archiv.

Um Simon zu installieren, ziehen Sie einfach das ausgepackte Programm in Ihren Programme-Ordner. Falls Sie mögen, können Sie das Programm auch sonstwo auf Ihrer Festplatte ablegen.
