Filters window

The Filters list shows all of the available filters. You can add, change, remove, or pause filters from this view.

This icon indicates whether or not this filters is appropriate and available when the service and/or previous filters are unchanged, as set in the Filter Info pane.

Similarly, this icon indicates that the filter is available when the status is changed.

This icon shows that a filter should be used when the status from the service and/or previous filters is a failure.

Kind: This shows which filter plug-in is used by this filter. The filters use plug-ins to perform the specific action, e.g. extracting a block, finding text, etc.

Status: This indicates whether this filter is currently available or paused.

Name: This shows the name you gave for each filter.
