Test InfoServiceMySQL

The MySQL service is displayed when you choose MySQL in the Service pop-up menu in the Test Info pane.

This service will connect to a specified MySQL database and perform a query, to ensure it is fully operational.

Server: Enter a server domain name.

Port: Enter a port to connect on, or leave it blank to use the default.

Socket: Enter the socket to use, for example "MySQL.sock".

Database: Enter a database name.

Query: Enter a query to perform, for example "SELECT * FROM table".

This plug-in was written by Daniel Ellis .
