Introduction

Welcome to Simon!

Simon is the essential site monitoring tool for Mac OS X, and Dejal's flagship product. It checks servers for changes or failures, and notifies you via email, sound, speech, or other means. You can use it to track updated sites, and to alert you when an important server goes down or recovers. It can be used to monitor your own website and servers, track posts and new comments on your or friends' blogs, check for web mail, get notifications of updates to favorite news and entertainment websites, keep an eye on auctions, communicate with arbitrary servers, and many other uses.

Simon is very intuitive, so you can probably find out everything you need to know just by using and exploring the application. But if you prefer to read up about software before using it, or would like more information about some aspect, this help book is here to help.

You can read this help book from within Simon by selecting the Simon Help item from the Help menu, or you can go directly to the page discussing a specific window by clicking the button in that window.

These help pages can also be read online via any web browser , if you prefer. See the choices in the Help menu to easily toggle all help options between these two locations.

We hope that you enjoy using Simon!
