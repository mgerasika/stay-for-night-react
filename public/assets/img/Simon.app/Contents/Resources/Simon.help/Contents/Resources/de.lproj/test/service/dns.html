Test-InfobereichDienstDNS

Dies ist der Dienst-Abschnitt "DNS". Er wird gezeigt, wenn Sie im Dienst-Pop-up-Menü im Test-Infobereich den Menüpunkt "DNS" wählen.

Dieser Dienst versucht, einen Domänen-Namen (aus einer IP) aufzulösen, so dass dieser Computer weiß, wo die Domäne ist.

Anders als der Dienst Spezifischer DNS prüft dieser Dienst nur den Standard-DNS-Server Ihres Macs. Wenn Sie einen anderen Server nutzen möchten, benutzen Sie stattdessen den Dienst "Spezifischer DNS".

Domänen-Name:
Geben Sie hier einen Domänen-Namen ein, der gesucht werden soll. Dieser Dienst prüft sehr schnell, ob es den eingegebenen Namen in eine IP-Adresse auflösen kann. Das bedeutet noch nicht, dass der entsprechende Server oder die Adresse momentan auch verfügbar sind, sondern nur, dass er weiß, wo der Server ist (oder wo er den Informationen des Domänen-Namen-Servers entsprechend sein soll).
