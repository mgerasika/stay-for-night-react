Dienst (Neu/Bearbeiten)Dienst-ArtDNS-Dienst-Modul

Das DNS-Modul wird gezeigt, wenn Sie im Dienst-Art-Pop-up-Menü im Dienst-Infobereich den Menüpunkt "DNS" wählen.

Dieses Modul wird vom Standard-Dienst DNS benutzt. Es prüft einen Domänen-Namen, um sicherzustellen, dass er in eine IP-Adresse aufgelöst werden kann, um so einen DNS-Server-Fehlschlag entdecken zu können.

Anders als der Dienst Spezifischer DNS prüft dieser Dienst nur den Standard-DNS-Server Ihres Macs. Wenn Sie einen anderen Server nutzen möchten, benutzen Sie stattdessen den Dienst "Spezifischer DNS".

Standard-Domäne:
Geben Sie den Standard-Domänen-Namen ein oder lassen Sie das Eingabefeld leer. Normalerweise bleibt das Feld leer.
