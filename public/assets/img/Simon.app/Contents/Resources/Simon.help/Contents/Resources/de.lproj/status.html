Status-Menü

Wenn in den Allgemein-Einstellungen die Option zum Anzeigen des Status in der Menüleiste aktiviert ist, bindet Simon ein Menü wie das dargestellte im oberen rechten Bereich der Menüleiste ein.

Das Menütitel-Icon zeigt den wichtigsten Status für alle nicht pausierenden Tests. Wenn irgendwelche Webseiten nicht erreichbar sind, wird das rote, abwärts zeigende Dreieck dargestellt. Wenn irgendeine Seite kürzlich Änderungen hatte, wird das grüne, nach oben zeigende Dreieck gezeigt. Ansonsten werden das orange Dreieck und eine Ziffer gezeigt, falls Seiten kürzlich Fehler hatten. Wie in der Testliste zeigt die Färbung des Dreiecks, wie lange die Statusänderung zurückliegt. Falls irgendwelche Tests als Ungesehen markiert sind, wird über dem Status-Dreieck ein gelber Sonnenschein-Kreis gezeigt, der die Anzahl ungesehener Tests zeigt. (Sie können dies in den Allgemein-Einstellungen abschalten, falls gewünscht). Das Status-Menü dient so als nützlicher visueller Hinweis auf wichtige Ereignisse.

Wenn Sie auf das Menütitel-Icon klicken, wird ein Menü gezeigt, das einige nützliche Kommandos enthält, die Sie direkt ausführen können, ohne Simon erst aktivieren zu müssen, wenn Sie gerade ein anderes Programm benutzen:

Simon-Monitor zeigen:
Dies zeigt das Simon-Monitor-Fenster. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Dienste zeigen:
Dies zeigt das Dienste-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Filter zeigen:
Dies zeigt das Filter-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Melder zeigen:
Dies zeigt das Melder-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Berichte zeigen:
Dies zeigt das Berichte-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Alle jetzt prüfen:
Dieses Kommando bewirkt, dass Simon augenblicklich alle Tests prüft, die nicht pausieren. Simon wird nicht aktiviert.
Alle Tests anhalten:
Wenn Sie dieses Kommando wählen, werden alle Tests angehalten und pausieren, so als würden Sie im Menüpunkt "Ablage > Test anhalten" die Option "Unendlich" wählen, wenn keine Tests selektiert sind. Simon wird nicht aktiviert.
Alle Tests fortsetzen:
Dieses Kommando macht das Gegenteil des vorigen, indem es für alle Tests die Pause wieder beendet. Simon wird nicht aktiviert.
Alle als Gesehen markieren:
Dieses Kommando markiert alle Tests, als wenn Sie sie schon gesehen hätten, d.h., der gelbe Marker wird aus allen Tests entfernt. Simon wird nicht aktiviert.

Danach werden alle Ihre Tests mit ihren Status aufgelistet, und zwar in der gleichen Reihenfolge wie im Simon-Monitor-Window. Wenn Sie den Mauszeiger über einen Test im Menü halten, zeigt er einen Hilfstext wie in der obigen Abbildung, mit Verfügbarkeit, Dienst und interessanten Zeitintervalle für diesen Test.

Wenn Sie einen Test im Status-Menü auswählen, wird die Aktion ausgeführt, die in den Allgemein-Einstellungen eingestellt ist. Wenn aber die Aktion "Untermenü anzeigen" dort eingestellt ist, wird ein Untermenü angezeigt, das weitere Optionen bietet, mit denen Sie einzelne Tests von Simon steuern können, ohne das Programm aktivieren zu müssen.

Dieses Untermenü enthält die folgenden Befehle: Test wählen, Test bearbeiten, Jetzt prüfen, Quelltext zeigen, Vorschau, Zur Website, Pause ein/aus und Betrachtet ein/aus, die woanders detailliert beschrieben sind.
