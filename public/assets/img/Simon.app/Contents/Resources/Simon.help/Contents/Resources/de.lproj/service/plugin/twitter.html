Dienst (Neu/Bearbeiten)Dienst-ArtTwitter-Dienst-Modul

Das Twitter-Dienst-Modul wird gezeigt, wenn Sie im Dienst-Art-Pop-up-Menü im Dienst-Infobereich den Menüpunkt "Twitter" wählen.

Dieses Modul wird vom Standard-Dienst Twitter benutzt. Hiermit können Sie Twitter-Updates, Direkt-Nachrichten, Freunde, Anhänger und mehr nach Änderungen und/oder Fehlschlägen beobachten.

Quelle:
Wählen Sie die Art der Twitter-Information, die standardmäßig geprüft werden sollen, wenn Sie einen Twitter-Test hinzufügen. Sie legen hier nur einen Standard-Vorschlag fest; beim Anlegen eines entsprechenden Tests können Sie ihn jederzeit ändern.

Dieses Modul wurde von Daniel Ellis geschrieben.
