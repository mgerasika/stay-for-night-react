Das Auf-den-entfernten-Server-hochladen-Modul wird gezeigt, wenn Sie im Bericht-Infobereich im Bericht-Art-Menü "Auf den entfernten Server hochladen" gewählt haben.

Dieses Modul lädt den Bericht über FTP auf Ihren Webserver hoch. Das ist praktisch, wenn Sie den Bericht von anderen Computern sehen möchten oder ihn auch Ihren Kunden und Bekannten zur Verfügung stellen wollen.

Sichern als:
Geben Sie hier den Namen ein, unter dem der Bericht ausgegeben werden soll. Falls nötig, wird ein Ordner mit diesem Namen erzeugt.
Host:
Geben Sie hier den Namen Ihres Webservers an, genauso wie beim manuellen Hochladen von Dateien auf Ihren Webserver.
Benutzername:
Geben Sie hier den Benutzernamen ein, mit dem Sie sich auf Ihrem Webserver per FTP einloggen.
Passwort:
Geben Sie das entsprechende Password zum Einloggen auf Ihren Webserver hier ein.
Pfad:
Geben Sie hier den Verzeichnispfad ein, in den der Bericht hochgeladen werden soll. Denken Sie daran, alle benötigten Ordner mit anzugeben; z.B. beginnen die meisten Server mit einem Verzeichnis namens "public_html". Achten Sie darauf, dass der Bericht nicht bereits existierende wichtige Webseiten überschreiben kann.
Öffentlicher URL:
Dies ist ein URL zum Bericht, der auf den obigen Informationen basiert. Sie können ihn bearbeiten, falls er nicht richtig zusammengestellt wurde.
Los:
Klicken Sie diesen Button an, um den Bericht in Ihrem Webbrowser über den öffentlichen URL anzusehen. Bei neuen Berichten ist das natürlich ist das erst sinnvoll, nachdem Sie mit einem Klick auf [OK] einen neuen Bericht erzeugt haben.
