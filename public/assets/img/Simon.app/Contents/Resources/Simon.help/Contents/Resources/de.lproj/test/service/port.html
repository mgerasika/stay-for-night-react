Test-InfobereichDienstPort-basierte Dienste: IMAP, POP, SMTP, SSH usw.

Ein Abschnitt ähnlich diesem wird für die Dienste "E-Mail-Eingang (IMAP)", "E-Mail-Eingang (POP)", "E-Mail-Versand (SMTP)" und "SSH" benutzt. Sie alle basieren auf dem Port-Dienst-Modul. Dieser Abschnitt wird gezeigt, wenn Sie im Dienst-Pop-up-Menü im Test-Infobereich einen dieser Dienste wählen.

Alle diese Dienste bauen eine Verbindung zum angegebenen Mailserver auf und versuchen dann, sich mit ihm zu unterhalten, basierend auf einem etablierten Skript, das bestimmte Antworten erwartet. Wenn Simon dieses Skript erfolgreich durchlaufen hat, wird die Prüfung als Erfolg gewertet. Falls Simon aber keine Verbindung herstellen kann, nichts empfängt, nicht die richtigen Antworten empfängt oder eine erkannte Fehler-Situation gemeldet bekommt, wird die Prüfung als Fehlschlag bewertet (und der Grund als Fehlermeldung protokolliert).

Domäne oder IP-Adresse:
Geben Sie hier den Domänen-Namen oder die IP-Adresse des Mailservers an, mit dem Sie eine Verbindung herstellen wollen.
