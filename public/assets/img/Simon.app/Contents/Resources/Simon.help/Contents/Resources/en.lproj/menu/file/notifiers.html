File menu for Notifiers list

The File menu for the Notifiers list contains commands to perform actions on notifiers:

New Notifier...: This function displays the Notifier Info pane, described elsewhere.

Duplicate Notifier...: This function displays the Notifier Info pane using values from the selected notifier as a starting point.

Notify Now: This will activate the selected notifier, making it perform its action. Useful to test it. The notification is not logged against any specific test.

Pause Notifier: This function displays a Pause Notifier sheet, similar to the Pause Test one. This is useful if you want to avoid certain notifications, without having to remove them from all of the tests.
