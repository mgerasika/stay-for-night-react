Ablage-Menü für das Filter-Liste

Das Ablage-Menü für das Filter enthält Kommandos, um Aktionen auf Dienste anzuwenden:

Neuer Filter:
Diese Funktion zeigt das Filter-Infobereich, das woanders beschrieben wird.
Filter duplizieren:
Diese Funktion zeigt das Filter-Infobereich window using values from the selected filter as a starting point.
Filter bearbeiten:
Diese Funktion zeigt das Filter-Infobereich, das woanders beschrieben wird.
Filter löschen:
Diese Funktion zeigt das Filter löschen, das woanders beschrieben wird.
Filter anhalten:
Diese Funktion zeigt das Formular "Filter anhalten", ähnlich dem für Test anhalten. Dies ist nützlich, wenn Sie bestimmte Filter nicht verwenden möchten, ohne dass gleich alle Tests mit pausieren müssen, die diese Filter verwenden.

Verwandte Themen

  • Filter (Neu/Bearbeiten)
  • Filter löschen
  • Filter anhalten
