Test-InfobereichOptionen

Dies ist der "Test-Optionen"-Abschnitt im Test-Infobereich. In diesem Abschnitt können Sie neben anderen Optionen einstellen, wie oft dieser Test geprüft werden soll. Nachfolgend werden die Optionen detailliert beschrieben:

Nach Erfolg neu prüfen in:
Wählen Sie die gewünschten Einheiten vom Pop-up-Menü (zur Auswahl stehen Sekunden, Minuten, Stunden, Tage, Wochen und Monate) und geben Sie den entsprechenden Wert im Eingabefeld ein. Wenn das so angegebene Zeitintervall abgelaufen ist, prüft Simon den Test automatisch erneut, wenn der vorige Test erfolgreich war. Damit sich Prüfung und Wartezeit nicht überlappen, ist das kleinste effektive Intervall etwa 20 Sekunden. Der Standardwert sind 15 Minuten.
Nach Fehlschlag neu prüfen in:
Diese Parameter werden genau wie die vorigen benutzt, aber sie legen fest, wie oft nach einem Fehlschlag geprüft werden soll. Vermutlich wollen Sie nach einem Fehler häufiger prüfen, um sehen zu können, wie schnell der Fehler behoben ist. Der Standardwert sind 5 Minuten.
Zeitüberschreitung nach:
Hier können Sie festlegen, wie lange eine Prüfung maximal dauern darf. Wenn sie nicht in der hier vorgegebenen Zeit abgeschlossen ist, wird das als Fehlschlag gewertet. Der Standardwert sind 3 Minuten.
Test auslassen, wenn alle geprüft werden:
Wenn Sie den Menüpunkt "Ablage > Jetzt prüfen" wählen oder den entsprechenden Button in der Symbolleiste anklicken, ohne dass ein Test selektiert ist, werden alle Tests auf einmal geprüft. Wenn Sie aber diese Option in einem Test abgehakt haben, wird der spezifische Test nicht automatisch mit geprüft. Das ist z.B. dann nützlich, wenn Sie eine langsame Internetverbindung und viele Tests haben und eine bestimmte Website lange Antwortzeiten hat.
Kann offline prüfen:
Wenn diese Option nicht abgehakt ist, kontrolliert Simon vor jeder Prüfung, ob der Computer mit dem Internet verbunden ist und gibt dem Test einen "Offline"-Status (mit einem Blitz-Symbol), falls keine Internetverbindung besteht. Aktivieren Sie diese Option, wenn Sie möchten, dass eine Trennung vom Internet als Fehlschlag behandelt wird oder wenn ein bestimmter Test gar keine Internetverbindung benötigt.
