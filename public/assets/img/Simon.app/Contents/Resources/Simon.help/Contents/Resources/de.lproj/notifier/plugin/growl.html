Melder-InfobereichMelder-ArtGrowl-Melder-Modul

Das Growl-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "Growl" wählen.

Dieses Modul benutzt den beliebten externen Growl -Benachrichtigungsdienst, der kleine Schwebefenster auf dem Bildschirm zeigt, um Meldungen auszugeben. Falls Sie ihn noch nicht installiert haben, zeigt dieses Modul einen Button, der die einfache Growl-Installation startet.

Variable einsetzen:
Variablen sind Marker im Text, die durch andere Werte ersetzt werden, wenn der Dienst benutzt wird. Wählen Sie ein Eingabefeld oder platzieren Sie den Eingabecursor in einem Feld. Anschließend wählen Sie in diesem Menü eine Variable, um sie in dem Feld einzufügen. Um zu erfahren, welche verschiedenen Variablen hier verfügbar sind, sehen Sie bitte im Kapitel Melder-Variablen nach.
Textfeld:
Geben Sie hier den Text ein, der in der Growl-Meldung ausgegeben werden soll, wenn dieser Melder benutzt wird. Sie können Variablen einfügen, die zwischen geschweiften Klammern stehen müssen. Wählen Sie Variablen direkt aus dem Menü "Variable einsetzen"; Experten können die Variablen direkt eingeben.
Priorität:
Wählen sie die Dringlichkeit der Meldung. Beachten Sie, dass nicht alle Growl-Vorlagen dies unterstützen.
Nicht ausblenden:
Wenn diese Option abgehakt ist, wird das Meldung-Fenster auf dem Bildschirm verbleiben, bis Sie es anklicken. Wenn Sie es anklicken, wird eine weitere Aktion ausgelöst. Wenn diese Option nicht gewählt ist, wird das Meldung-Fenster von Growl nach ein paar Sekunden ausgeblendet.
Klick-Aktion:
Für Meldungen, die auf dem Bildschirm verbleiben, könenn Sie hier festlegen, was passieren soll, wenn Sie das Growl-Meldung-Fenster anklicken. Zur Auswahl stehen Keine, Simon nach vorne bringen, Dock-Icon hüpfen lassen, Test in Simon selektieren, Vorschau und Zur Website.
