SimonUpdates suchen

Es ist wichtig, dass Sie Ihre Programme aktuell halten, weil immer wieder Fehler behoben, Funktionen verbessert oder hinzugefügt werden und andere nötige oder wichtige Erweiterungen eingebaut werden. Es kann sehr lästig sein, dazu von Hand Websites zu durchstöbern, daher haben wir für Sie in Simon die Sache einfacher gemacht. Simon kann nach aktualisierten Versionen des Programm suchen, entweder automatisch in bestimmten Intervallen, die Sie im Einstellungen-Fenster machen können, oder manuell, indem Sie den Menüeintrag Simon > Updates suchen wählen.

In den Einstellungen können Sie außerdem wählen, ob nur nach Haupt-Versionen oder auch nach Beta-Versionen gesucht werden soll. Standardmäßig wird nach dem Versionstyp gesucht, den Sie beim ersten Installieren des Programms genutzt haben: Wenn Sie mit einer Beta-Version anfingen, erhalten Sie Meldungen über folgende Betas und Haupt-Versionen; ansonsten werden nur Haupt-Versionen angeboten. Sie können dies jederzeit ändern.

Eine andere nützliche Option ist die Möglichkeit, Updates automatisch installieren zu lassen.

Wenn Simon nach Updates sucht, kontaktiert es den Dejal-Server (wenn Ihr Computer mit dem Internet verbunden ist) und erfährt dort, ob eine aktualisierte Version des Programms verfügbar ist. Während dieser Prüfung können Sie einfach weiterarbeiten, z.B. falls Sie eine langsame Internetverbindung haben (bei DSL-Verbindungen geht es blitzschnell).

Wenn eine neue Version verfügbar ist, erscheint ein Fenster, das Sie informiert, welche Version Sie momentan nutzen und zeigt die neue Version zusammen mit Versions-Informationen für die neue Version. Dabei zeigt es automatisch alle Versions-Informationen zwischen der aktuellen und Ihrer Version an. Falls gewünscht, können Sie hier ebenfalls ältere Versions-Informationen nachschlagen. Das Fenster zeigt ebenso einen Download-Button für die neue Version sowie einen Button, der zur Simon-Website führt, falls Sie weitere Informationen wünschen. Falls Sie Ihre Version momentan aber noch gar nicht aktualisieren wollen, schließen Sie das Fenster einfach wieder.

Falls Ihr Programm noch aktuell ist, öffnet sich bei manueller Update-Suche trotzdem ein Fenster, das allerdings anders aussieht. Es teilt Ihnen die gute Nachricht mit, dass Ihr Programm noch immer aktuell ist und zeigt andere Informationen vom Dejal-Server; in diesem Beispiel werden Ihnen andere Dejal-Produkte vorgestellt, die Sie vielleicht einmal ausprobieren wollen. Probieren kostet nichts!
