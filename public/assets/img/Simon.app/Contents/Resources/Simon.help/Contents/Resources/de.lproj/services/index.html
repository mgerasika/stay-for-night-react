Dienste-Liste

Das Dienste-Liste listet alle verfügbaren Dienste auf. Sie können in diesem Fenster Dienste hinzufügen, ändern, entfernen oder pausieren lassen.

Normalerweise werden Sie kaum nötig haben, die Dienste zu ändern. Simon bringt schon verschiedenste vorgefertigte Dienste mit, die Sie so benutzen können. Aber falls Sie einen bestimmten Dienst benötigen, den Sie beobachten wollen, können Sie im Dienste-Liste eigene Dienste hinzufügen. Oder falls Ihnen die Standardvorgaben der Dienste nicht zusagen, können Sie sie hier Ihren Bedürfnissen anpassen.

Art:
Diese Spalte listet auf, welche Art von Dienst-Modul dieser Dienst nutzt. Die Dienste nutzen Module, um spezifische Aktionen auszuführen, wie z.B. das Beobachten von Webseiten, lokalen Programmen oder Server-Ports.
Status:
Diese Spalte zeigt, ob der Dienst momentan verfügbar ist oder pausiert.
Name:
Diese Spalte zeigt den Namen, den Sie jedem Dienst gegeben haben.
