Test InfoFilterOverride

The Override filter is displayed when you choose Override in the Filter pop-up menu in the Test Info pane.

The default Override as Unchanged, Override Custom and Override No Fail filters are based on the Override filter plug-in.

You can use these filters to alter the status of a test from Unchanged, Changed, Failure, or from a success status (the first two), or any status, into a different status, including automatically detecting if a change occurred. Automatic detection works by comparing the output of this filter to the output the last time it was used by this test, resulting in a Changed status if different, otherwise Unchanged.

You can also alter the output text or error message, including using variables (similar to the Format plug-in).

Some filters may display controls for what to override, others may not display anything, if they are pre-configured to override a certain situation.
