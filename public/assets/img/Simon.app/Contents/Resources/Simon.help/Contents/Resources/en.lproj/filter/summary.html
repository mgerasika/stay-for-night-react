Filter InfoName

The Name page of the Filter Info pane contains fields to name the filter and write a description of it:

Name: Enter a descriptive name for the filter; you can use anything that helps you identify it; Simon does not use this value other than for display purposes. The default is "Untitled".

Comments: Enter a description of the filter, or comments about it. This is displayed in tooltips on the pop-up menu of filters in the Filters page of the Test Info pane, as a handy quick-reference when choosing filters. The default is blank.
