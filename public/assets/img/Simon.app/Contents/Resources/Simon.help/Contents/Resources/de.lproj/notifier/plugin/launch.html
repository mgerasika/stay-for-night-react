Melder-InfobereichMelder-ArtStarten-Melder-Modul

Das Starten-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "Starten" wählen.

Starten:
Dieses Pop-up-Menü enthält drei Möglichkeiten, um zu kontrollieren, was und wie gestartet werden soll:
  • Nur das Programm startet das festgelegte Programm oder wechselt zum festgelegten Programm ohne irgendein zugewiesenes Dokument. Das entspricht einem Doppelklick auf das entsprechende Programm-Icon im Finder.
  • Dokument mit Standard-Programm bedeutet, dass das festgelegte Dokument mit dem dafür im System festgelegten Programm geöffnet wird. Das entspricht einem Doppelklick auf das Dokument im Finder.
  • Dokument mit zugewiesenem Programm startet das festgelegte Programm oder wechselt zu diesem und öffnet das festgelegte Dokument in diesem Programm. Das entspricht dem Ziehen eines Dokuments auf ein Programm-Icon im Finder.
Programm:
Klicken Sie auf den [Wählen]-Button, um die Dateiauswahl aufzurufen, und wählen Sie dort ein Programm aus. Dessen Icon und Pfad werden anschließend hier gezeigt. Falls im Starten-Pop-up-Menü eine Option gewählt wurde, die kein festzulegendes Programm benötigt, ist dieser Bereich nicht wählbar.
Dokument:
Klicken Sie auf den [Wählen]-Button, um die Dateiauswahl aufzurufen, und wählen Sie dort ein Dokument aus. Dessen Icon und Pfad werden anschließend hier gezeigt. Falls im Starten-Pop-up-Menü eine Option gewählt wurde, die kein festzulegendes Dokument benötigt, ist dieser Bereich nicht wählbar.
