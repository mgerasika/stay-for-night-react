Service InfoSummary

The Summary page of the Service Info pane contains fields to name the service and write a description of it, plus an overview of values from other pages:

Name: Enter a descriptive name for the service; you can use anything that helps you identify it; Simon does not use this value other than for display purposes. The default is "Untitled".

Service Kind, etc: Information about the service is displayed, as a quick overview. Click the title before each item to quickly go to the corresponding page and control.

Comments: Enter a description of the service, or comments about it. This is displayed in tooltips on the pop-up menu of services in the Services page of the Test Info pane, as a handy quick-reference when choosing services. The default is blank.
