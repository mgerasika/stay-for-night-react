Filter InfoFilter KindOverride plug-in

The Override plug-in is displayed when you choose Override in the Filter Kind pop-up menu in the Filter Info pane.

This plug-in is used by the default Override as Unchanged, Override Custom, and Override No Fail filters.

You can use filters based on this plug-in to alter the status of a test from Unchanged, Changed, Failure, or from a success status (the first two), or any status, into a different status, including automatically detecting if a change occurred. Automatic detection works by comparing the output of this filter to the output the last time it was used by this test, resulting in a Changed status if different, otherwise Unchanged.

You can also alter the output text or error message, including using variables (similar to the Format plug-in).
