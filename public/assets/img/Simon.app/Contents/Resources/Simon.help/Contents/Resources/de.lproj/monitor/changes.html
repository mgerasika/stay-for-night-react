MonitorÄnderungen

Das Änderungen-Protokoll am unteren Rand des Simon-Monitor-Fensters enthält eine Tabelle, die die letzten Änderungen der Tests zeigt. Wie bei den Prüfungen gilt auch hier: Wenn keine Tests selektiert sind, werden die Änderungen aller Tests gelistet; wenn einige Tests oder nur ein Test selektiert sind, werden nur die Änderungen der selektierten Tests gelistet. Die Spalten der Tabelle können sortiert und waagerecht verschoben werden, falls gewünscht. Diese Tabelle ist hauptsächlich dafür interessant, wenn man sehen will, wie oft sich eine Website ändert oder wann sie in der Vergangenheit verändert wurde.

Auch dieser Bereich bietet ein Suchfeld. Nutzen Sie es, um in der Tabelle gezeigte Informationen weiter einzugrenzen. Selektieren Sie z.B. einen Test oder mehrere Tests in der Tests-Tabelle, geben Sie dann einen Text im Suchfeld ein, und Sie werden nur die die Tabellen-Zeilen sehen, die diesen Text enthalten.

Änderungen werden über die Filter-Funktion entdeckt. Eine beliebige Anzahl Filter kann sich den Ausgabetext des Dienstes ansehen und wahlweise prüfen, ob es seit der letzten Prüfung Änderungen gibt – Simon merkt sich immer den vorigen Ausgabetext für jeden Filter. Auf der Filter-Seite finden Sie hierzu weitere Informationen.

Hier sind die Details zu den einzelnen Spalten:

Änderungsdatum:
Diese Spalte zeigt, wann die Änderung aufgetreten ist, oder genauer gesagt, wann Simon die Website geprüft und die Änderung entdeckt hat. Dazu werden die Prüfkriterien benutzt, die Sie im Test-Fenster(Neu/Bearbeiten) eingestellt haben.
Test-Name:
Diese Spalte zeigt den Namen des Tests, dessen Änderungen hier aufgeführt werden; nützlich, um mehrere Tests unterscheiden zu können.
Unterschied:
Diese Spalte zeigt den neuen Text, der zwischen dem festgelegten Starttext und Endetext in der Prüfung gefunden wurde. Auch wenn der geänderte Text sehr lang ist, wird nur der Teil gezeigt, der tatsächlich geändert ist. Falls der neue Text sehr kurz ist, wird er komplett gezeigt (z.B. Zugriffszähler für Forum-Beiträge und ähnliches).
