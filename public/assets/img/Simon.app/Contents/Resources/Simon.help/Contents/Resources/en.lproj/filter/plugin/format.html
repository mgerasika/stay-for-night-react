Filter InfoFilter KindFormat plug-in

The Format plug-in is displayed when you choose Format in the Filter Kind pop-up menu in the Filter Info pane.

This plug-in is used by the default Simple Format filter.

This plug-in enables the combination of values from the service and filters, via an insert variable menu, along with your own text. Handy to append the output of multiple filters, wrap something in quote marks, and other uses.

This filter outputs the specified text, with variables replaced by their values. It can detect changes via the checkbox in the Options page.
