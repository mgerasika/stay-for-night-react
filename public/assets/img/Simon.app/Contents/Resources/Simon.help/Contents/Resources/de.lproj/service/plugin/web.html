Dienst (Neu/Bearbeiten)Dienst-ArtWeb-Dienst-Modul

Das Web-Dienst-Modul (für HTTP und FTP) wird gezeigt wenn Sie im Dienst-Art-Pop-up-Menü im Dienst-Infobereich den Menüpunkt "Web-Dienst (HTTP/FTP)" wählen.

Dieses Modul wird vom Standard-Dienst Web (HTTP) und von FTP-Diensten genutzt. Es prüft eine Website oder einen FTP-Server auf Änderungen oder Fehlschläge.

Standard-Protokoll:
Wählen Sie das Protokoll für den Dienst: http://, https://, feed://, ftp:// oder file://. Wenn Sie einen URL ohne Protokoll eingeben (z.B. "www.dejal.com" oder nur "dejal"), wird das hier angegebene Standard-Protokoll verwendet.
Standard-URL:
Wenn Sie möchten, dass jeder neue HTTP- oder FTP-Dienst einen bestimmten URL standardmäßig prüft, geben Sie ihn hier ein. Ansonsten lassen Sie das Eingabefeld leer. Standardmäßig ist das Feld leer.
Automatisch .www und .com automatisch hinzufügen, falls nötig:
Wenn diese Option abgehakt ist und ein URL im Fenster "Test bearbeiten" ohne Top-Level-Domäne eingegeben wird, wird die Eingabe mit "www." und ".com" umschlossen. Wenn Sie z.B. nur "apple" eingeben, erhalten Sie "www.apple.com". Standardmäßig ist diese Option eingeschaltet; schalten Sie sie aus, wenn Sie diese automatische Vervollständigung nicht wünschen.
