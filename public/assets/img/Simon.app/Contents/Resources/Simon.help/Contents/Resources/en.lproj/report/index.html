Report Info pane

The Report Info pane is accessed when the Reports list is active. It shows a summary of the report; click the heading for each line of the summary to jump right to that page. It includes an Edit button to enable modifying the report, which toggles to Done when editing to switch back to the summary.

In Edit mode, the Info pane is divided into four pages. Below is a brief description of each; click on the page name for more information about each:

Name: This page includes fields where you can name this report and write comments about it.

Options: This page lets you specify how often this report should be output, what pages to generate, what template to use, etc.

Report Kind: Choose the report plug-in on this page. Simon uses plug-ins to do the actual output, allowing greater flexibility and expandability.

Auto Pause: This feature allows you to suspend this report on specified days (or any day) and times. Useful to avoid outputting reports when you know nobody will view them.
