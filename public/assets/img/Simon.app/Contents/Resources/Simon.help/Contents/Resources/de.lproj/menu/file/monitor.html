Ablage-Menü für das Simon-Monitor-Fenster

Das Ablage-Menü für das Simon-Monitor-Fenster enthält Kommandos, um Aktionen auf Tests anzuwenden:

Neuer Test:
Diese Funktion zeigt das Fenster Neuer Test, das woanders beschrieben wird.
Test duplizieren:
Diese Funktion zeigt das Fenster Neuer Test, das schon mit Werten des selektierten Tests als Ausgangspunkt vorbelegt ist.
Test bearbeiten:
Diese Funktion zeigt das Fenster Test bearbeiten, das woanders beschrieben wird.
Test löschen:
Diese Funktion zeigt das Formular Test löschen, das woanders beschrieben wird.
Sichern:
Diese Funktion ändert sich je nach Status der gewählten Protokoll-Ansicht. Es sichert die aktuellen Protokoll-Informationen in eine tab-getrennte Textdatei. Wenn mindestens zwei Protokollzeilen selektiert sind, werden nur die selektierten Zeilen gesichert, ansonsten alle Zeilen. (Sie können zudem selektierte Zeilen kopieren.)
Jetzt prüfen:
Im Simon-Monitor-Fenster bewirkt dieses Kommando, dass Simon umgehend die selektierten Tests prüft, anstatt zu warten, bis sie das nächste Mal an der Reihe sind. Der Zeitpunkt für die nächste Prüfung wird anschließend zurückgesetzt auf der Basis des Zeitpunkts, an dem die Prüfung gestartet wurde. Sie können diese Funktion nutzen, um die Prüf-Zeit relativ zu anderen Tests zu ändern, wenn Sie möchten. Wenn keine Tests selektiert sind, werden alle Tests auf einmal geprüft, aber die nächste Prüf-Zeit wird nicht zurückgesetzt. Im Quelltext-Fenster und in den Vorschau-Fenstern bewirkt es, das die aktuelle Seite erneut vom Server geladen und neu dargestellt wird.
Zwischengespeicherte neu laden:
Dieses Kommando wird nur für das Quelltext-Fenster und die Vorschau-Fenster verwendet. Es lädt die Seite erneut aus dem Puffer, d.h., ohne sie erneut vom Server zu laden. Dabei wird die Seite neu dargestellt und alle Bilder geladen und neu gerendert.
Quelltext zeigen:
Dieses Kommando öffnet ein neues Quelltext-Fenster für den selektierten Test. Lesen Sie die Beschreibung dieses Fensters für weitere Informationen.
Vorschau:
Diese Funktion zeigt ein neues Vorschau-Fenster für den selektierten Test; auch dieses Fenster ist woanders beschrieben.
Zur Website:
Dieses Kommando öffnet Ihren Standard-Webbrowser mit dem URL des selektierten Tests. Dies ist beispielsweise dann nützlich, wenn Simon Sie informiert, dass die Seite sich geändert hat, so dass Sie die Änderungen direkt sehen können.
Statistiken löschen:
Diese Funktion zeigt das Formular Statistiken löschen, das woanders beschrieben wird.
Test anhalten:
Diese Funktion zeigt das Formular Test anhalten, das woanders beschrieben wird.

Verwandte Themen

  • Test-Infobereich
  • Vorschau zeigen
  • Test löschen
  • Test zurücksetzen
  • Test anhalten
