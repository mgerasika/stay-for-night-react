Filter-InfobereichFilter-ArtÜberschreiben-Filtermodul

Das Überschreiben-Filtermodul wird gezeigt, wenn Sie im Filter-Art-Pop-up-Menü im Filter-Infobereich "Überschreiben" wählen.

Dieses Modul wird standardmäßig von den Filtern "Als Unverändert ausgeben", "Benutzerdefinierte Änderung" und "Nicht als Fehlschlag ausgeben" genutzt.

Sie können Filter verwenden, die auf diesem Modul basieren, um den Status eines Tests von Unverändert, Geändert, Fehlschlag oder von einem Erfolg-Status (die ersten beiden) oder von irgendeinem Status in einen anderen Status zu ändern, wobei automatisch erkannt wird, ob eine Änderung stattfand. Die automatische Erkennung vergleicht den Ausgabetext dieses Filters mit dem vorigen Ausgabetext dieses Filters bei diesem Test, was zum Geändert-Status führt, wenn sie unterschiedlich sind, ansonsten zu Unverändert.

Sie können ebenfalls den Ausgabetext oder die Fehlermeldung ändern, wozu sogar Variablen verwendet werden können (ähnlich wie im Formatieren -Modul).
