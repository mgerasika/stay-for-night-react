Service InfoService KindApplication plug-in

The Application plug-in is displayed when you choose Application in the Service Kind pop-up menu in the Service Info pane.

This plug-in is used by the default Application service. It will check for the presence of a local application on your computer, and optionally whether or not it is currently running. It's useful to use this in conjunction with the Launch notifier, to launch it if it isn't running.

Default Path: Click the Set button to display an open panel, and choose a default path. This is used as the starting point when this service is used.
