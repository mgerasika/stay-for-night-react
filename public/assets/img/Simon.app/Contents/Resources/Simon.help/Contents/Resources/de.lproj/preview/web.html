Vorschau-Fenster für Webseiten (HTTP)

Öffnen Sie das Quelltext-Fenster, indem Sie einen Test anklicken und dann den Menüpunkt "Ablage > Vorschau" wählen oder den [Vorschau]-Button in der Symbolleiste anklicken. Dies ist das Vorschau-Fenster für den Webseiten-(HTTP)-Dienst; andere Dienste benutzen ein etwas anderes Fenster, wenn sie eine Vorschau unterstützen.

Dieses Fenster ist nützlich, um eine rasche Vorschau einer Webseite zu bekommen, einschließlich aller Bilder. Die Vorschau zeigt alles exakt wie ein Webbrowser, dank der Safari-Web-Routinen. Im oberen Textbereich enthält das Fenster zudem den HTML-Quelltext für Vergleichszwecke, wenn Sie Text für die Filter-Funktion lokalisieren wollen. Wie im Quelltext-Fenster gibt es auch hier einen Schiebebalken zwischen den beiden Textbereichen, der es ermöglicht, die Proportionen der Textbereiche anzupassen. An der Unterkante des Fensters werden Datum und Zeitpunkt, an dem die Seite gepuffert wurde, der Inhaltstyp und der Ergebnis-Code gezeigt, sowie ein Fortschrittsbalken, der den Ladevorgang der Seite anzeigt.

URL:
Hier wird der Ort der Seite oder Datei angezeigt, dessen Quelltext hier angesehen werden kann. Sie können einen neuen URL direkt in diesem Feld eingeben, wenn Sie möchten; drücken Sie [Return], um die neue Seite zu laden. Wenn der URL gepuffert wurde, wird die zwischengespeicherte Kopie geladen, ansonsten wird die Seite frisch vom Server geholt (und in den Puffer kopiert).
Quelltext:
Dieser Textbereich zeigt den HTML-Quelltext der Seite, wie er vom Webserver ausgegeben wurde.
Seite:
Dieser Textbereich zeigt die grafische Präsentation der Webseite, einschließlich aller Texte und Bilder. Sie können die Hyperlinks anklicken, um andere Seiten zu laden, genau wie man es von einem Browser erwartet.
