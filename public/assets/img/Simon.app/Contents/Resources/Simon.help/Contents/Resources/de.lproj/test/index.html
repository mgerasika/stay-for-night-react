Test-Infobereich

Die Fenster "Neuer Test" und "Test bearbeiten". Dieses Fenster wird üblicherweise über die Menüpunkte "Ablage > Neuer Test" bzw. "Ablage > Test bearbeiten" oder die entsprechenden Buttons der Symbolleiste geöffnet.

Das Fenster ist in sieben Seiten unterteilt. Nachfolgend werden alle kurz beschrieben; klicken Sie auf einen Seitennamen, um mehr Informationen zu erhalten:

Zusammenfassung:
Die erste Seite enthält Felder, in denen Sie diesen Test benennen und Kommentare hinzufügen können, sowie eine Übersicht der anderen Seiten.
Optionen:
In dieser Seite könnten neben anderen Optionen einstellen, wie oft dieser Test geprüft werden soll.
Dienst:
Diese Seite enthält ein Pop-up-Menü, das die verfügbaren Dienste auflistet. Wenn Sie einen Dienst aus diesem Menü auswählen, ändern sich die Felder darunter entsprechend.
Filter:
In dieser Seite können Sie beliebig viele Filter hinzufügen, die nach Änderungen suchen oder den Ausgabetext filtern. Dies ist eine sehr nützliche Funktion; klicken Sie auf den Link, um weitere Informationen zu erhalten.
Melder:
In dieser Seite können Sie beliebig viele Melder aktivieren, die Änderungen, Fehlschläge oder Reparaturen melden, damit Sie immer informiert werden, wenn eines der eingestellten Ereignisse eintritt.
Berichte:
In dieser Seite können Sie festlegen, ob dieser Test in allen Berichten aufgenommen wird, nur in einigen oder in keinem Bericht.
Auto-Pause:
Diese Funktion macht es möglich, die Test-Prüfungen an bestimmten Tagen (oder allen Tagen) oder zu bestimmten Zeiten zu pausieren. Sehr praktisch, wenn z.B. ein beobachteter Server immer um Mitternacht neu gestartet wird, oder wenn Sie genau wissen, dass ein Server bestimmte Zeiten hat, an denen er nicht benutzt wird usw.
Quelltext zeigen:
Wenn Sie diesen Button anklicken, wird ein Quelltext-Fenster für den aktuellen Test geöffnet. Der Button ist nur anklickbar, wenn das Dienst-Modul diese Funktion unterstützt.
Vorschau:
Klicken Sie diesen Button an, um ein Vorschau-Fenster für Webseiten (HTTP) oder ein Allgemeines Vorschau-Fenster für den aktuellen Test zu öffnen. Der Button ist nur anklickbar, wenn das Dienst-Modul diese Funktion unterstützt.
