Bericht-InfobereichZusammenfassung

Die Zusammenfassung-Seite im Bericht-Infobereich enthält Felder, um den Test zu benennen und eine Beschreibung zu notieren, sowie eine Übersicht der Werte aus den anderen Seiten:

Name:
Geben Sie dem Test einen sinnvollen Namen; nutzen Sie alles, was Ihnen hilft, den Test später rasch wiederzufinden. Er erscheint sowohl im Bericht-Fenster als auch auf der Zusammenfassung-Seite des generierten Berichts (wenn die {title}-Variable genutzt wurde).
Aktualisieren usw.:
Hier sehen Sie eine Kurzübersicht über die Bericht-Parameter. Klicken Sie den Titel vor jedem Eintrag an, um rasch die entsprechende Seite aufzurufen, wenn Sie die Parameter kontrollieren wollen.
Kommentar:
Geben Sie hier eine Beschreibung des Berichts oder Kommentare ein. Der Text wird in Hilfstexten im Pop-up-Menü von Diensten in der Berichte-Seite des Fensters "Test bearbeiten" als bequeme Kurzreferenz angezeigt, wenn Sie Berichte auswählen. Standardmäßig ist das Feld leer.
