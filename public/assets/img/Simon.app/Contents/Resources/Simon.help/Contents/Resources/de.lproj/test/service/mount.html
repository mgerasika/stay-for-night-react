Test-InfobereichDienstAktivieren

Dies ist der Dienst-Abschnitt "Aktivieren". Er wird gezeigt, wenn Sie im Dienst-Pop-up-Menü im Test-Infobereich den Menüpunkt "Aktivieren" wählen.

Dieser Dienst versucht, ein lokales, per AFP oder SMB verbundenes Laufwerk zu aktivieren, um zu prüfen ob es verfügbar ist. Anschließend kann der Dienst das Laufwerk optional wieder deaktivieren.

Protokoll:
Wählen Sie das Protokoll für das Laufwerk.
Server:
Geben Sie hier den Servernamen für das AFP- oder SMB-Laufwerk ein.
Laufwerk:
Geben Sie hier den gemeinsam benutzten Laufwerk-Namen ein.
Deaktivieren, wenn fertig:
Haken Sie diese Option ab, wenn Sie möchten, dass nach der Prüfung das Laufwerk wieder deaktiviert wird, oder lassen Sie die Option ungewählt, wenn das Laufwerk aktiviert bleiben soll.

Dieses Modul wurde von Daniel Ellis geschrieben.
