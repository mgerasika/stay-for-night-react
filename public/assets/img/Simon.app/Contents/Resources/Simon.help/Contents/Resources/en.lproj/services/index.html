Services list

The Services list shows all of the available services. You can add, change, remove, or pause services from this view.

Normally you probably won't need to modify the services. Simon comes with several pre-defined services, so you can just use them as-is. However, if you have a special type of service you want to monitor, the Services window allows you to add your own. Or if the default values for the services aren't to your liking, you can customize them to suit you.

Kind: This lists which service kind plug-in is used by this service. The services use plug-ins to perform the specific action, e.g. checking a website, a local application, or a server port.

Status: This indicates whether this service is currently available or paused.

Name: This shows the name you gave for each service.
