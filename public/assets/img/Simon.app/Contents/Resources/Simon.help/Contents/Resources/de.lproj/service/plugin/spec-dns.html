Dienst (Neu/Bearbeiten)Dienst-ArtSpezifischer DNS-Dienst-Modul

Das Spezifischer-DNS-Dienst-Modul wird gezeigt, wenn Sie im Dienst-Art-Pop-up-Menü im Dienst-Infobereich den Menüpunkt "Spezifischer DNS" wählen.

Dieses Modul wird vom Standard-Dienst Spezifischer DNS benutzt. Es prüft für einen Domänen-Namen, ob er in eine IP-Adresse aufgelöst werden kann, um so festzustellen, ob ein DNS-Server-Fehlschlag vorliegt.

Gesuchte Domäne:
Geben Sie den Standard-Domänen-Namen ein oder lassen Sie das Eingabefeld leer. Normalerweise bleibt das Feld leer.
Nameserver-Adresse:
Geben Sie hier den DNS-Server ein, der zur Prüfung des Domänen-Namens genutzt werden soll.
IP-Prüfung:
Wählen Sie hier "Jedes Resultat akzeptieren", um den Test als Erfolg zu werten, wenn der DNS-Service irgendeine gültige IP-Adresse zurückgibt, oder wählen Sie "Resultat muss hierzu passen", wenn es ein Fehlschlag sein soll, sobald eine andere IP-Adresse zurückgegeben wird als die angegebene.
Rekursive Abfragen unterbinden:
Wenn diese Option abgehakt ist, wird nur der angegebene DNS-Server befragt; ansonsten werden auch andere DNS-Server akzeptiert.

Dieses Modul wurde von Daniel Ellis geschrieben.
