Report TemplatesFailure

This page has information about the Failure block, and variables valid within this scope. See the Report Templates page for more information about Simon's report templates.

The Failure block includes variables about a failure - i.e. values displayed in Simon's Monitor window in the failures log.

Variables are tags in the file that are replaced by another value when the report is used. Variables are enclosed in curly braces, and have a variable name optionally followed by any number of parameters, e.g. {failed type="relative"}. See the parameters page for information on the available parameters.

The variables that are available in this scope include:

Variable:Outputs:
DowntimeThe length of time between failure and recovery.
ErrorThe error message returned when performing the check.
FailedThe date and time that the test failed.
RecoveredThe date and time that the test recovered.
