Ablage-Menü für das Berichte-Liste

Das Ablage-Menü für das Berichte-Liste enthält Kommandos, um Aktionen auf Berichte anzuwenden:

Neuer Bericht:
Diese Funktion zeigt das Bericht-Infobereich, das woanders beschrieben wird.
Bericht duplizieren:
Diese Funktion zeigt das Bericht-Infobereich, das schon mit Werten des selektierten Berichts als Ausgangspunkt vorbelegt ist.
Bericht bearbeiten:
Diese Funktion zeigt das Fenster "Bericht bearbeiten", das woanders beschrieben wird.
Bericht löschen:
Diese Funktion zeigt das Formular Bericht löschen, das woanders beschrieben wird.
Jetzt berichten:
Diese Funktion gibt die selektierten Berichte aus, oder alle Berichte, falls keine selektiert sind.
Bericht-Vorschau:
Zeigt den selektierten Bericht in Simons Vorschau-Fenster.
Bericht ansehen:
Zeigt den selektierten Bericht in Ihrem Webbrowser.
Berichtspuffer leeren:
Diese Funktion zeigt das Formular Berichtspuffer leeren, das woanders beschrieben wird.
Bericht anhalten:
Diese Funktion zeigt das Formular "Bericht anhalten", ähnlich dem für Test anhaltenSie ist nützlich, wenn Sie für eine Weile keine Berichte erhalten wollen.

Verwandte Themen

  • Bericht-Infobereich
  • Bericht löschen
  • Bericht zurücksetzen
  • Bericht anhalten
