Ablage-Menü für das Melder-Liste

Das Ablage-Menü für das Melder-Liste enthält Kommandos, um Aktionen auf Melder anzuwenden:

Neuer Melder:
Diese Funktion zeigt das Fenster Neuer Melder, das woanders beschrieben wird.
Melder duplizieren:
Diese Funktion zeigt das Fenster Neuer Melder, das schon mit Werten des selektierten Melders als Ausgangspunkt vorbelegt ist.
Melder bearbeiten:
Diese Funktion zeigt das Fenster Melder bearbeiten, das woanders beschrieben wird.
Melder löschen:
Diese Funktion zeigt das Formular Melder löschen, das woanders beschrieben wird.
Jetzt melden:
Hiermit wird der selektierte Melder aktiviert, so dass er seine Aktion ausführt. Nützlich, um ihn zu testen. Die Meldung wird nicht für einen spezifischen Test protokolliert.
Melder anhalten:
Diese Funktion zeigt das Formular "Melder anhalten", ähnlich dem für "Test anhalten". Dies ist nützlich, wenn Sie bestimmte Melder nicht verwenden möchten, ohne dass gleich alle Tests mit pausieren müssen, die diese Dienste verwenden.
