Test InfoServiceScript-based services

A section similar to this is used for the FTP Directory Listing service, and others based on the Script service plug-in. This is displayed when you choose one of those services in the Service pop-up menu in the Test Info pane.

These services all have a description followed by zero or more fields, representing custom variables for the script. The Script plug-in is a very powerful feature, allowing you and others to add an unlimited number of new services to Simon, that leverage the power of AppleScripts, shell scripts, and other script languages. See the Script service plug-in for more information.

You can get more scripts, and share ones you create, via the Simon Extras page on the Dejal website. Check it out!
