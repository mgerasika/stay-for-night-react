MonitorStatistiken löschen

Das Formular "Statistiken löschen" wird im Simon-Monitor-Fenster gezeigt, wenn Sie den Menüpunkt "Ablage > Statistiken löschen" wählen oder den Button [Statistiken löschen] in der Symbolleiste anklicken (der standardmäßig nicht in der Symbolleiste ist).

Wenn Sie das Löschen bestätigen, werden die Statistiken für die selektierten Tests gelöscht, wobei alle Prüfungen, Änderungen, Fehlschläge und Aktivitäten für diese Tests gelöscht werden.

Nach dem Löschen der Statistiken wird für die selektierten Tests-Tabelle angezeigt, wann ihre Statistiken gelöscht wurden, bis ein neues Ereignis eintritt. So können Sie nachverfolgen, wie lange die letzte Änderung oder der letzte Fehlschlag zurückliegt.

Diese Funktion ist nur verfügbar, wenn mindestens ein Test selektiert ist. Entsprechend weist der Text in diesem Formlar darauf hin, dass Sie keinen, einen oder mehrere Tests selektiert haben, damit Sie nicht aus Versehen die Statistiken für mehr Tests löschen als beabsichtigt. Wenn keine Tests selektiert sind, bezieht sich diese Funktion auf alle Tests.
