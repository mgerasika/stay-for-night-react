Das Bericht-lokal-speichern-Modul wird gezeigt, wenn Sie im Bericht-Infobereich im Bericht-Art-Menü "Lokal speichern" gewählt haben.

Dieses Modul sichert den Bericht in einen Ordner auf Ihrem Computer. Das ist sinnvoll, wenn Sie Persönliches Web-Sharing nutzen möchten, um Berichte auf Ihrem Computer bereitzustellen, oder wenn Sie die Berichte selbst von Hand irgendwo hochladen wollen (z.B. nur für gelegentlichen Gebrauch), oder wenn Sie eine Nicht-HTML-Vorlage wie "Tab-Delimited Text" nutzen.

Sichern als:
Geben Sie hier den Namen ein, unter dem der Bericht ausgegeben werden soll. Falls nötig, wird ein Ordner mit diesem Namen erzeugt.
Wo:
Wählen Sie einen Standard-Ort auf Ihrem Computer, oder wählen Sie "Andere", um einen eigenen Ort zu wählen.
Pfad:
Hier wird der Ausgabe-Pfad angezeigt. Sie müssen hier nichts eingeben, aber Sie können den Pfad bearbeiten, falls Sie z.B. weitere Unterordner nutzen oder einen alternativen Zugang zum Berichts-Ordner haben wollen.
Zugang über:
Dies ist ein URL zum Bericht. Klicken Sie ihn an, um den Bericht in Ihrem Webbrowser zu sehen.
