Notifier InfoName

The Name page of the Notifier Info pane contains fields to name the notifier and write a description of it:

Name: Enter a descriptive name for the notifier; you can use anything that helps you identify it; Simon does not use this value other than for display purposes. The default is "Untitled".

Comments: Enter a description of the notifier, or comments about it. This is purely for your own reference. The default is blank.
