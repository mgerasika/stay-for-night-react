Willkommen zu Simon!

Simon ist das unerlässliche Überwachungswerkzeug für Mac OS X und gleichzeitig das Hauptprodukt von Dejal Systems. Es prüft Server auf Änderungen oder Fehlschläge und benachrichtigt Sie via E-Mail, Ton, Sprache oder andere Methoden. Benutzen Sie das Programm, um Änderungen an Websites zu beobachten oder um Warnmeldungen zu erhalten, wenn ein wichtiger Server ausfällt oder wieder hochfährt. Sie können mit Simon auch Ihre eigenen Webseiten und Server überwachen, die Blogs Ihrer Freunde auf neue Nachrichten und Kommentare prüfen lassen, nachsehen, ob neue Web-Mails vorliegen oder benachrichtigt werden, wenn Ihre Lieblings-Nachrichtenseiten oder Unterhaltungs-Websites aktualisiert wurden. Oder lassen Sie Simon ein Auge auf Ihre Auktionen haben, sich mit beliebigen Servern unterhalten und vieles mehr.

Simon ist sehr intuitiv bedienbar, daher werden Sie vermutlich alles, was Sie wissen möchten, rasch selbst herausfinden, wenn Sie das Programm nutzen und entdecken. Aber wenn Sie es vorziehen, erst über eine Software zu lesen, bevor Sie sie benutzen, oder falls Sie mehr Informationen über bestimmte Aspekte haben möchten, hilft Ihnen dieses Handbuch weiter.

Sie können dieses Handbuch direkt in Simon lesen, indem Sie im Hilfe-Menü den Eintrag Simon-Hilfe wählen. Oder Sie rufen direkt eine Seite im Handbuch auf, die ein bestimmtes Programm-Fenster erklärt, indem Sie im Programm-Fenster auf den -Button klicken.

Dieses Handbch können Sie ebenfalls online über jeden Webbrowser erreichen , wenn Sie mögen. Im Hilfe-Menü finden Sie Möglichkeiten, für alle Hilfe-Optionen zwischen diesen beiden Orten zu wechseln.

Wenn Sie irgendwelche Verbesserungsvorschläge haben, etwas im Handbuch nicht gut erklärt finden oder sonstige Nachrichten für uns haben, lassen Sie es uns bitte wissen, damit wir das Programm und/oder das Handbuch verbessern können.

Wir hoffen, dass Sie viel Freude mit Simon haben.

Um loszulegen, rufen Sie die Hilfe-Homepage oder den Index auf und wählen Sie ein Thema.
