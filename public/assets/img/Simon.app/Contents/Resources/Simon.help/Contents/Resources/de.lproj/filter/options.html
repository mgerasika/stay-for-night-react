Filter-InfobereichOptionen

In der "Optionen"-Seite im Filter-Infobereich können Sie festlegen, für welche Kategorie dieser Filter genutzt werden soll, und zwar für denn Fall, dass der aktuelle Status des Tests (d.h., des Dienstes und/oder aller vorherigen Filter) ein Erfolg oder Fehlschlag ist.

Unverändert:
Haken Sie diese Option ab, wenn dieser Filter wirken soll, wenn der Status unverändert ist. Anschließend erscheint er im Pop-up-Menü der Filter-Seite im Test-Infobereich, so dass er gewählt werden kann, wenn der aktuelle Status des Tests unverändert ist.
Verändert:
Wählen Sie diese Option, wenn dieser Filter bei verändertem Status wirken soll, ansonsten wie oben beschrieben. Um den Filter verfügbar zu machen, wenn der Test erfolgreich war (d.h., kein Fehlschlag), haken Sie beide Optionen ab.
Fehlschlag:
Ähnlich der o.g. Checkbox zeigt dies an, dass dieser Filter für Fehlschläge genutzt werden kann, z.B. wenn eine Website nicht mehr erreichbar ist.
Im Eingabetext nach Änderungen suchen:
Wenn diese Option gewählt ist, wird der Filter seinen Ausgabetext mit dem vom letzten Mal vergleichen, als er für diesen Test benutzt wurde. Wenn er einen Unterschied feststellt, gibt er den Status Geändert zurück, oder Unverändert, wenn beide identisch sind. Wenn diese Option nicht gewählt ist, wird immer Unverändert zurückgegeben (so lange kein Fehler auftritt).
Erlaubt, das Test-Bearbeiten-Fenster einzustellen:
Wenn diese Option abgehakt ist, erscheinen in der "Filter"-Seite im Test-Infobereich Kontrollmöglichkeiten zum Einstellen des Filters. Wenn nicht, werden keine Kontrollmöglichkeiten eingeblendet, so dass immer die Einstellung aus diesem Fenster genutzt wird. Damit können Sie im Vergleich zu den allgemein gehaltenen Filtern sehr spezielle Filter erstellen, z.B. um den Body einer Webseite zu finden.
Zeitüberschreitung nach:
Legen Sie hier fest, nach welcher Zeitspanne dieser Filter seine Aufgabe erledigt haben muss. (Die meisten Filter sind augenblicklich fertig, aber skriptbasierte Filter können schon mal länger dauern.)
