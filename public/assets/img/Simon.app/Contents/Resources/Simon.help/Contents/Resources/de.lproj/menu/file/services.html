Ablage-Menü für das Dienste-Liste

Das Ablage-Menü für das Dienste-Liste enthält Kommandos, um Aktionen auf Dienste anzuwenden:

Neuer Dienst:
Diese Funktion zeigt das Dienst-Infobereich, das woanders beschrieben wird.
Dienst duplizieren:
Diese Funktion zeigt das Dienst-Infobereich, das schon mit Werten des selektierten Dienstes als Ausgangspunkt vorbelegt ist.
Dienst bearbeiten:
Diese Funktion zeigt das Dienst bearbeiten, das woanders beschrieben wird.
Dienst löschen...:
Diese Funktion zeigt das Dienst löschen, das woanders beschrieben wird.
Dienst anhalten:
Diese Funktion zeigt das Formular "Dienst anhalten", ähnlich dem für Test anhalten. Dies ist nützlich, wenn Sie bestimmte Dienste nicht verwenden möchten, ohne dass gleich alle Tests mit pausieren müssen, die diese Dienste verwenden.

Verwandte Themen

  • Dienst-Infobereich
  • Dienst löschen
  • Dienst anhalten
