MonitorStatistiken

Der Statistiken-Bereich des Simon-Monitor-Fensters zeigt interessante Statistiken zu den Tests. Abhängig davon, ob Sie einen einzelnen Test, mehrere oder alle (bzw. keine) Tests selektiert haben, wird eine Zusammenfassung für Ihre Auswahl an Tests gezeigt. Wenn Sie mit gedrückt gehaltener []-Taste mehrere Tests selektiert haben oder mit []-Klick einen Bereich markiert haben, zeigen die Statistiken eine Zusammenfassung für die selektierten Tests. Wenn nur ein Test selektiert ist, werden die Statistiken für genau diesen einen Test dargestellt. So können Sie sehr rasch einen Überblick über Ihre Tests erhalten und sich auf einzelne oder einen bestimmten Test konzentrieren.

Erfolge und Fehlschläge:
Dieser Bereich zeigt die zusammengefasste Verfügbarkeit der selektierten Tests (oder aller Tests, falls keiner selektiert ist), die Gesamtanzahl der durchgeführten Prüfungen, die Anzahl der erfolgreichen und fehlgeschlagenen Prüfungen und die Anzahl der Tests, die einen Fehlschlag hatten und anschließend gerettet wurden.
Prüfdauer:
In diesem Bereich sehen Sie die kürzeste, längste und durchschnittliche Zeit, die für die Durchführung der Tests benötigt wurde. Ebenso wird hier angezeigt, wie lange die zuletzt durchgeführte Prüfung dauerte. So können Sie z.B. einen Bereich und die durchschnittliche Antwortzeit eines Servers beobachten.
Zeit zwischen Fehlschlägen:
Dieser Bereich zeigt die kürzeste, längste und durchschnittliche Zeit zwischen Fehlschlägen. Genauer gesagt, zeigt er die Zeit zwischen Rettungen nach Fehlschlägen.
