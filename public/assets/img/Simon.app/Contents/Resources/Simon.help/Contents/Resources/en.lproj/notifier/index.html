Notifier Info pane

The Notifier Info pane is accessed when the Notifiers list is active. It shows a summary of the notifier; click the heading for each line of the summary to jump right to that page. It includes an Edit button to enable modifying the notifier, which toggles to Done when editing to switch back to the summary.

In Edit mode, the Info pane is divided into four pages. Below is a brief description of each; click on the page name for more information about each:

Name: This page includes fields where you can name this notifier and write comments about it.

Options: This page lets you specify how this notifier should be categorized: for changes, for failures, for recoveries, or some combination. Some notifiers are useful for all, but some (e.g. that have text saying a site is down) are only appropriate for a specific type of event.

Notifier Kind: Choose the notifier plug-in on this page. Simon uses plug-ins to provide the actual notification functionality, allowing greater flexibility and expandability.

Auto Pause: This feature allows you to suspend this notifier on specified days (or any day) and times, without stopping the check from occurring. Useful to avoid receiving notifications at inconvenient times, like when you're asleep.
