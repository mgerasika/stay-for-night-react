Help menu

The items in this menu will take you to the relevant pages on the Dejal website. When you choose one of these commands, your default web browser (as specified in your System Preferences) is activated and the web page is loaded. You can simulate each here by clicking the titles below:

Simon Help: After the standard help search field, this item displays the help book either in Apple's standard help viewer or your web browser , depending on the following setting.

Read Help in Help Viewer: Choose this option to read the help in Apple's standard help viewer.

Read Help in Browser: Choose this option if you prefer to read the same help content in your web browser. The Simon Help item above, plus all buttons, will show the help in your browser instead of the help viewer.

Simon Website : This command will open your web browser to the main page describing the Simon product.

Simon Release Notes : This will take you to the Release Notes for the version of Simon that you are currently running, which details what was fixed or added in that version. At the top of that page is an indication whether that version is current or old, so this is useful as an alternative to the SimonCheck for Updates feature. (Note, the link here redirects you to the Release Notes for the latest release, since this page has no way of knowing which version you are using. The command in Simon will intelligently direct you to the appropriate Release Notes.)

Simon Extras : This will take you to the Simon Extras page, that contains updated and user-submitted scripts, report templates, and more.

Simon Frequently Asked Questions : Have a question? This is the page to visit first! The Dejal FAQ (Frequently Asked Questions) page answers many common questions about the company, purchasing our products, and product-specific questions.

Simon Support Forum : Displays the Simon Forum. This is the preferred place to suggest features, report bugs, or discuss other comments about Simon.

Post MacUpdate Review : Help spread the word about Simon! Posting a review on MacUpdate really helps encourage others to give it a try.

Post iusethis Review : iusethis.com is another great software listing site; posting a review there too also helps.

Tweet About Simon on Twitter : Prepares a tweet on Twitter for you to tell your followers about Simon (doesn't actually send it).

Share About Simon on Facebook : Prepares a post on Facebook for you to tell your friends about Simon (doesn't actually post it).

Dejal on Twitter: @dejal : Displays the @dejal Twitter account, where news about Simon and other apps (and more) is posted. Follow @dejal to get updates in your timeline!

Simon on Twitter: @DejalSimon : Displays the @DejalSimon Twitter account, where Simon expresses thoughts about itself. Follow @DejalSimon to get updates in your timeline!

Dejal on Facebook : Displays the Dejal Facebook page, where news about Simon and other apps (and more) is posted. "Like" the Dejal page to get updates in your timeline!

Dejal Website : This command will direct your browser to the main Dejal page. This page has product announcements, company news, and links to other interesting areas of the Dejal site. Reload the page if necessary to ensure that you see the latest news.

Dejal Blog : Displays the Dejal Blog, where products are announced and discussed, along with general Mac commentary and much more.

Dejal Newsletter : Subscribe to the Dejal newsletter to receive very occasional news updates via email.

Dejal Store : This page is where you can purchase our software.
