Test-InfobereichZusammenfassung

Die Zusammenfassung-Seite im Test-Infobereich enthält Felder, um den Test zu benennen und eine Beschreibung zu notieren, sowie eine Übersicht der Werte aus den anderen Seiten:

Name:
Geben Sie dem Test einen sinnvollen Namen; nutzen Sie alles, was Ihnen hilft, den Test später rasch wiederzufinden; Simon nutzt diesen Namen nur für Anzeige-Zwecke. Der Standard ist "Unbenannt".
Frequenz, usw.:
Hier sehen Sie eine Kurzübersicht über die Test-Parameter. Klicken Sie den Titel vor jedem Eintrag an, um rasch die entsprechende Seite aufzurufen, wenn Sie die Parameter kontrollieren wollen.
Kommentar:
Geben Sie hier eine Beschreibung des Tests oder Kommentare ein. Dieses Feld steht nur Ihnen zur Verfügung. Standardmäßig ist das Feld leer.
