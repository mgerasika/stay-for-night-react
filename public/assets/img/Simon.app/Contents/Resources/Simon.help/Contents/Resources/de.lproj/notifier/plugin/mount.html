Melder-InfobereichMelder-ArtAktivieren-Dienst-Modul

Das Aktivieren-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "Aktivieren" wählen.

Dieses Modul aktiviert oder deaktiviert ein lokales, AFP- oder SMB-Laufwerk.

Protokoll:
Wählen Sie das Protokoll für das Laufwerk.
Bei Meldung:
Wählen Sie, ob bei Meldung das Laufwerk aktiviert oder deaktiviert werden soll.
Server:
Geben Sie hier den Servernamen für das AFP- oder SMB-Laufwerk ein.
Laufwerk:
Geben Sie hier den Namen des gemeinsam genutzten Laufwerks ein.
Benutzername:
Geben Sie hier den Benutzernamen für das Laufwerk ein.
Passwort:
Geben Sie hier das Passwort für das Laufwerk ein.

Dieses Modul wurde von Daniel Ellis geschrieben.
