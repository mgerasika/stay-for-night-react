Test-InfobereichDienstMySQL

Dies ist der Dienst-Abschnitt "MySQL". Er wird gezeigt, wenn Sie im Dienst-Pop-up-Menü im Test-Infobereich den Menüpunkt "MySQL" wählen.

Dieser Dienst stellt eine Verbindung zu einer definierten MySQL-Datenbank her und führt eine Anfrage durch, um sicherzustellen, dass die Datenbank voll funktionstüchtig ist.

Server:
Geben Sie hier den MySQL-Server-Namen ein oder lassen Sie das Feld leer. Normalerweise sollten Sie das Feld leer lassen.
Port:
Geben Sie hier den Port ein, über den die Verbindung laufen soll oder lassen Sie das Feld leer, um den Standard-Port zu nutzen. Normalerweise sollten Sie das Feld leer lassen.
Socket:
Geben Sie den zu nutzenden Socket ein, z.B. "MySQL.sock".
Datenbank:
Geben Sie hier den Namen der Datenbank ein.
Abfrage:
Geben Sie hier eine durchzuführende Abfrage ein, z.B. "SELECT * FROM table".
