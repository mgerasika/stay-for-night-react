File menu for Reports list

The File menu for the Reports list contains commands to perform actions on reports:

New Report...: This function displays the Report Info pane, described elsewhere.

Duplicate Report...: This function displays the Report Info pane using values from the selected report as a starting point.

Report Now: This function outputs the selected report(s), or all reports if none are selected.

Preview Report: Displays the selected report in Simon's Preview pane.

View Report: Displays the selected report in your web browser.

Reset Reports Cache...: This function displays the Reset Reports Cache confirmation sheet, described elsewhere.

Pause Report...: This function displays a Pause Report sheet, similar to the Pause Test one. This is useful if you want to stop reporting for a while.
