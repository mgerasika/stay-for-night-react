Test-InfobereichFilterScript-based filters

Eine ähnliche Seite wie diese wird für Filter benutt, die auf dem Skript-Filtermodul. basieren. Sie wird gezeigt, wenn Sie einen dieser Filter im Filter-Pop-up-Menü im Test-Infobereich wählen.

Diese Filter zeigen alle zunächst eine Beschreibung, anschließend folgen kein oder mehr Eingabefelder, in denen selbstdefinierte Variablen des Skripts eingegeben werden müssen. Das Skript-Filtermodul ist eine sehr leistungsfähige Funktion, die allen Simon-Anwendern ermöglicht, Simon beliebig viele neue Dienste hinzuzufügen, die die Stärken von AppleScript, Shell-Skripten und anderen Skript-Sprachen nutzen können. Schauen Sie unter Skript-Filtermodul nach weiteren Informationen hierzu.

Auf der Simon-Extras -Seite auf der Dejal-Website finden Sie weitere Skripte und können selbsterstellte Skripte mit anderen teilen. Probieren Sie es aus!
