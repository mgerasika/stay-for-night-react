MonitorTest löschen

Das Formular "Test löschen" wird im Simon-Monitor-Fenster gezeigt, wenn Sie den Menüpunkt "Ablage > Test löschen" wählen oder den Button [Löschen] in der Symbolleiste anklicken.

Wenn Sie die Löschung bestätigen, wird der selektierte Test gelöscht. Dies kann nicht rückgängig gemacht werden.

Diese Funktion ist nur verfügbar, wenn mindestens ein Test selektiert ist. Entsprechend weist das Formular "Test löschen" darauf hin, dass Sie keinen, einen oder mehrere Tests löschen wollen, damit Sie nicht aus Versehen mehr Tests löschen als beabsichtigt.
