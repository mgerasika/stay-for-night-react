Service InfoService KindSpecific DNS plug-in

The Specific DNS plug-in is displayed when you choose Specific DNS in the Service Kind pop-up menu in the Service Info pane.

This plug-in is used by the default Specific DNS service. It will check a domain name to make sure it can be resolved to an IP address, to detect a DNS server failure.

Name to lookup: Enter a default domain name, or leave it blank. Normally you'd leave the default blank.

Name server address: Enter a DNS server to use to check for the domain name.

IP Checking: Select Accept any result to treat the test as a success if the DNS server returns any valid IP address, or Result must match to treat it as a failure if the domain resolves to an IP other than the specified one.

Disable recursive queries: If checked, only the specified DNS server is queried; otherwise others are allowed to weigh in too.

This plug-in was written by Daniel Ellis .
