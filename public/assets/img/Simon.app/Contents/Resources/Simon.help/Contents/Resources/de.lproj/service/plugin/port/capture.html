Das Sitzung-erfassen-Fenster für den Port-Dienst und Port-Melder-Module wird gezeigt, wenn Sie in der Dienst- oder Melder-Seite der entsprechenden Module auf den Button [Sitzung erfassen] klicken.

Dieses Fenster hilft Ihnen, das Sitzungs-Skript zu erstellen, indem es auf einem angegebenen Port mit einem Server eine Verbindung herstellt und Ihnen hilft, eine Unterhaltung mit diesem Server zu führen. Im oberen Bereich sehen Sie den Transkript-Bereich, der alle Nachrichten vom und zum Server listet. Darunter sind Felder, um eine Verbindung zu einem Server herzustellen, Text zu empfangen und Kommandos zu senden:

Domäne oder IP:
Geben Sie einen Domänen-Namen oder eine IP-Adresse ein, um auf dem angegebenen Port damit eine Verbindung herzustellen, und klicken Sie dann auf [Verbinden]. Falls eine gültige Domäne diesen Port unterstützt, wird sie die Verbindung zulassen; diese Adresse ist nur dazu da, das Sitzungs-Skript zu erfassen, nicht für spätere Prüfungen.
Verbinden:
Klicken Sie auf diesen Button, nachdem Sie einen Domänen-Namen oder eine IP-Adresse eingegeben haben. Simon versucht anschließend, eine Verbindung zu dem Server herzustellen. Falls er Erfolg hat, ändert der Button seinen Text zu [Trennen], so dass Sie mit einem weiteren Klick die Unterhaltung mit dem Server beenden und das Fenster schließen können. Das Fenster wird ebenfalls geschlossen, wenn der Server von sich aus die Verbindung trennt.
Empfangen:
Wenn Simon vom Server irgendwelchen Text empfängt, wird er sowohl im Transkript-Bereich als auch in diesem Feld dargestellt. Sie können dann den relevanten Teil des Textes in diesem Feld selektieren; "relevant" meint hier einen Text, von dem Sie ausgehen, dass der Server ihn in ähnlichen erfolgreichen Dialogen ausgeben wird. Wenn Sie den nächsten Befehl senden, wird dieser Text im "Wenn empfangen"-Feld eines neuen Sitzungs-Skriptes gespeichert.
Senden:
Geben Sie hier den nächsten Befehl ein, um das Skript fortzusetzen, und klicken Sie den nebenstehenden Button, um den Befehl zum Server zu senden. Anschließend sollten Sie einen Befehl senden, um die Operation zu beenden, z.B. "quit" oder etwas entsprechendes, und Simon schließt das Fenster, sobald der Server die Verbindung getrennt hat. Wenn nichts davon zutrifft, klicken Sie einfach auf [Trennen] oder schließen Sie das Fenster.
