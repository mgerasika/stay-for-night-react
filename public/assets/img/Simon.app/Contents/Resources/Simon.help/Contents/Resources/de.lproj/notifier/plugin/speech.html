Melder-InfobereichMelder-ArtSprachausgabe-Melder-Modul

Das Sprachausgabe-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich "Sprechen" wählen.

Variable einsetzen:
Variablen sind Marker im Text, die durch andere Werte ersetzt werden, wenn der Dienst benutzt wird. Wählen Sie ein Eingabefeld oder platzieren Sie den Eingabecursor in einem Feld. Anschließend wählen Sie in diesem Menü eine Variable, um sie in dem Feld einzufügen. Um zu erfahren, welche verschiedenen Variablen hier verfügbar sind, sehen Sie bitte im Kapitel Melder-Variablen nach.
Textfeld:
Geben Sie hier den Text ein, der gesprochen werden soll, wenn dieser Melder benutzt wird. Sie können Variablen einfügen, die zwischen geschweiften Klammern stehen müssen. Wählen Sie Variablen direkt aus dem Menü "Variable einsetzen"; Experten können die Variablen direkt eingeben.
Stimme:
Dies ist die Liste verfügbarer Stimmen. Jede Stimme hat bestimmte Eigenschaften, die durch die Schieberegler verändert werden können. Manche Stimmen haben eine sehr hohe Qualität und klingen wirklich menschlich, während andere eher mechanisch oder ziemlich schräg klingen.
Rate:
Dieser Schieberegler kontrolliert die Sprechgeschwindigkeit. Sie können den Regler nach links oder rechts schieben; weiter nach links verlangsamt die Sprechgeschwindigkeit, weiter rechts wird schneller gesprochen.
Tonhöhe:
Dieser Schieberegler kontrolliert, wie hoch oder tief die Stimme klingt. Hier gilt: weiter links ist die Stimme tiefer, weiter rechts ist sie höher.
Flexion:
Dieser Schieberegler kontrolliert, wie abwechslungsreich die Stimme ist. Wenn Sie ihn ganz nach links schieben, klingt die Stimme sehr roboterhaft, ohne irgendwelche Aussprachedynamik. Wenn Sie den Regler ganz nach rechts schieben, klingt die Stimme eher aufregend, mit vielen dynamischen Variationen. Irgendwo in der Mitte finden Sie sicher eine Ihnen zusagende Aussprachedynamik für die gewählte Stimme.
Lautstärke:
Der Lautstärke-Schieberegler steuert, wie laut die Stimme ist. Standardmäßig ist der Schieberegler ganz rechts, so dass die normale Lautstärke genutzt wird. Schieben Sie den Schieberegler weiter nach links, so wird die Stimme immer leiser, bis hin zu einem Flüstern.
Sprechen:
Klicken Sie diesen Button an, um sich den eingegebenen Text mit den aktuellen Stimmen-Einstellungen vorsprechen zu lassen.
Standard einstellen:
Mit diesem Button setzen Sie die vier Schieberegler auf ihre Standardpositionen für diese Stimme zurück. Sehr praktisch, wenn Sie nach Experimenten wieder zur normalen Stimmen-Einstellung zurückkehren möchten.
