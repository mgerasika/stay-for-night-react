DiensteDienst löschen

Das Formular "Dienst löschen" wird im Dienste-Liste gezeigt, wenn Sie den Menüpunkt "Ablage > Dienst löschen" wählen oder den Button [Löschen] in der Symbolleiste anklicken.

Wenn Sie die Löschaktion bestätigen, werden die gewählten Dienste gelöscht. Dies kann nicht rückgängig gemacht werden.

Diese Funktion ist nur verfügbar, wenn mindestens ein Dienst selektiert ist. Entsprechend weist das Formular "Dienst löschen" darauf hin, dass Sie keinen, einen oder mehrere Dienste löschen wollen, damit Sie nicht aus Versehen mehr Dienste löschen als beabsichtigt.

Wenn ein Dienst momentan von einem Test benutzt wird, erscheint ein Formular entsprechend dem oben gezeigten. Sie müssen dann den Test oder die Tests ändern oder löschen, die diesen Dienst benutzen, bevor Sie ihn löschen können.
