Dock-Symbol und Menü

Simon kann wichtige Informationen im "Bildschirm" seines Programm-Icons im Dock von Mac OS X zeigen. Wenn das Programm läuft und einige Tests hinzugefügt wurden, zeigt der Bildschirm-Bereich des Icons ein Status-Symbol wie in der Testliste.

Wenn irgendwelche Webseiten nicht erreichbar sind, wird das rote, abwärts zeigende Dreieck dargestellt, das die Anzahl der nicht erreichbaren Seiten als Ziffer zeigt. Wenn irgendeine Seite kürzlich Änderungen hatte, wird das grüne, nach oben zeigende Dreieck gezeigt, entsprechend mit einer Ziffer, die die Anzahl der geänderten Seiten zeigt. Ansonsten werden das orange Dreieck und eine Ziffer gezeigt, falls Seiten kürzlich Fehler hatten. Wie in der Testliste zeigt die Färbung des Dreiecks, wie lange die Statusänderung zurückliegt. Falls irgendwelche Tests als "Ungesehen" markiert sind, wird hinter dem Status-Dreieck ein gelber Sonnenschein-Kreis gezeigt (den Sie in den Allgemein-Einstellungen abschalten können, falls gewünscht). Das Status-Menü dient so als nützlicher visueller Hinweis auf wichtige Ereignisse.

Falls irgendwelche Tests als "Ungesehen" markiert sind, wird über dem Status-Dreieck ein gelber Sonnenschein-Kreis gezeigt, mit der Anzahl ungeprüfter Tests. Sie können dies in den Allgemein-Einstellungen abschalten, falls gewünscht.

Nach einem Rechtsklick (oder [Ctrl]-Klick) auf das Programm-Icon im Dock erscheint ein Menü, wie unter Mac OS X üblich. Zusätzlich zu den üblichen Menüeinträgen enthält das Menü einige nützliche Kommandos, die Sie direkt ausführen können, ohne Simon erst aktivieren zu müssen, wenn Sie gerade ein anderes Programm benutzen:

Simon-Monitor zeigen:
Dies zeigt das Simon-Monitor-Fenster. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Dienste zeigen:
Dies zeigt das Dienste-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Filter zeigen:
Dies zeigt das Filter-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Melder zeigen:
Dies zeigt das Melder-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Berichte zeigen:
Dies zeigt das Berichte-Liste. Simon wird aktiviert, wenn es nicht das aktuelle Programm war.
Alle jetzt prüfen:
Dieses Kommando bewirkt, dass Simon augenblicklich alle Tests prüft, die nicht pausieren. Simon wird nicht aktiviert.
Alle Tests anhalten:
Wenn Sie dieses Kommando wählen, werden alle Tests angehalten und pausieren, so als würden Sie im Menüpunkt "Ablage > Test anhalten" die Option "Unendlich" wählen, wenn keine Tests selektiert sind. Simon wird nicht aktiviert.
Alle Tests fortsetzen:
Dieses Kommando macht das Gegenteil des vorigen, indem es für alle Tests die Pause wieder beendet. Simon wird nicht aktiviert.
Alle als Gesehen markieren:
Dieses Kommando markiert alle Tests, als wenn Sie sie schon gesehen hätten, d.h., der gelbe Marker wird aus allen Tests entfernt. Simon wird nicht aktiviert.
