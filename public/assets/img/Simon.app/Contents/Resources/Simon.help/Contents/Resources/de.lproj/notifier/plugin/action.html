Melder-InfobereichMelder-ArtAktion-Melder-Modul

Das Aktion-Melder-Modul wird gezeigt, wenn Sie im Melder-Art-Pop-up-Menü im Melder-Infobereich Aktion wählen.

Dieses Modul bietet verschiedene Checkboxen. Sie können verschiedene von ihnen gleichzeitig wählen, um verschiedene Aktionen auf einmal durchzuführen, wenn dieser Melder benutzt wird.

Den Test in Simons Monitorfenster selektieren:
Wählen Sie diese Option, damit nur die Zeile des entsprechenden Tests selektiert wird, mit den dazugehörenden Änderungen in den Statistiken und Tabellen für Prüfungen, Änderungen, Fehlschläge und Aktivitäten.
Das Dock-Icon einmal hüpfen lassen, wenn Simon nicht das aktive Programm ist:
Wenn diese Checkbox abgehakt ist, hüpft das Simon-Icon im Dock einmal kurz, wenn dieser Melder benutzt wird, damit Sie darauf aufmerksam werden. Aber das Hüpfen wird nicht wiederholt. Diese und die folgende Checkbox schließen einander gegenseitig aus.
Das Dock-Icon hüpfen lassen, bis Simon nach vorne gebracht wurde:
Wenn diese Checkbox gewählt ist, hüpft das Simon-Icon im Dock so lange, bis das Simon-Programm nach vorn gebracht wird (durch einen Klick auf das Icon oder in eines der Fenster von Simon). Diese und die vorige Checkbox schließen einander gegenseitig aus.
Simons Monitorfenster vor allen anderen Simon-Fenstern zeigen:
Wenn diese Checkbox gewählt ist, wird das Simon-Hauptfenster vor alle anderen Simon-Fenster gebracht, z.B. vor die Vorschau-Fenster, aber es wird nicht zum vordersten Fenster, wenn Simon momentan nicht das aktive Programm ist. Diese und die folgende Checkbox schließen einander gegenseitig aus.
Simons Monitorfenster vor allen anderen Programmen zeigen:
Ist diese Option gewählt, wird das Simon-Hauptfenster ganz nach vorn gebracht, egal welches gerade das aktive Programm ist. Simon wird dadurch zum aktiven Programm. Diese und die vorige Checkbox schließen einander gegenseitig aus.
Das Quelltext-Fenster für den Test zeigen:
Wählen Sie diese Option, um das Quelltext-Fenster für den entsprechenden Test zu zeigen, wenn der Melder benutzt wird. Simon wird nicht zum aktiven Programm, falls es das noch nicht ist – falls Sie das Quelltext-Fenster vor allen anderen Programmen sehen wollen, haken Sie zusätzlich noch die vorige Checkbox ab, damit das Simon-Monitor-Fenster vor allen anderen Programmen liegt.
Die Seitenvorschau für den Test zeigen:
Diese Option ist ähnlich der vorigen Checkbox, bezieht sich aber auf das Vorschau-Fenster.
Die Seite im Web-Browser zeigen:
Diese Option entspricht der Funktion "Zur Website": Wenn ein Melder mit dieser Option benutzt wird, wird die entsprechende Seite in Ihrem bevorzugten Webbrowser-Programm geöffnet. Sehr praktisch, wenn Sie eine Webseite immer gleich sehen wollen, sobald sich dort etwas geändert hat.
