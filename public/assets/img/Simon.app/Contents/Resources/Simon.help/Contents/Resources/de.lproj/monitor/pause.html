MonitorTest anhalten

Das Formular "Test anhalten" wird im Simon-Monitor-Fenster gezeigt, wenn Sie den Menüpunkt "Ablage > Test anhalten" wählen oder den Button [Pause] in der Symbolleiste wählen. Wählen Sie die gewünschte Zeiteinheit aus dem Pop-up-Menü und geben Sie den gewünschten Wert im Eingabefeld ein. Das Pop-up-Menü bietet folgende Optionen: Minuten, Stunden, Tage, Wochen, Monate und unendlich. Simon merkt sich alle Zeitintervalle, die Sie zuvor benutzt haben, so dass Sie ganz leicht einen Test nochmal für die selbe Dauer pausieren lassen können.

Wenn Sie die Pause bestätigen, werden die selektierten Tests nicht mehr geprüft, bis der festgelegte Zeitraum abgelaufen ist. Die restliche Pausenzeit wird in der Tests-Tabelle (in der Spalte "Nächste Prüfung") angezeigt. Wenn die Pausenzeit abgelaufen ist, werden die entsprechenden Tests automatisch wieder fortgesetzt. Wenn Sie allerdings unendlich gewählt haben, bleiben die entsprechenden Tests im Pause-Modus, bis Sie sie von Hand wieder freigeben.

Diese Funktion ist nur verfügbar, wenn mindestens ein Test selektiert ist. Entsprechend weist der Text in diesem Formular darauf hin, dass Sie keinen, einen oder mehrere Tests selektiert haben. Wenn keine Tests selektiert sind, bezieht sich diese Funktion auf alle Tests.

Wenn Tests angehalten sind, können Sie die Option [Test fortsetzen] wählen, um die Pause aufzuheben. Dazu ist keine Bestätigung nötig.

Notiz: Simon kann Tests auch für eine bestimmten Zeitraum mit Hilfe der Funktion Auto-Pause anhalten.
