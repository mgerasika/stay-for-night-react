Test-InfobereichBerichte

Dies ist der Abschnitt "Berichte" im Test-Infobereich. In diesem Abschnitt können Sie festlegen, ob dieser Test in allen Berichten, in keinem Bericht oder nur in bestimmten Berichten mit aufgeführt wird:

Berichte:
Wählen Sie den Menüpunkt "In alle Berichte einschließen", wenn Sie möchten, dass dieser Test in allen Berichten erscheint, die Sie anlegen. Der Menüpunkt "Aus allen Berichten ausschließen" stellt sicher, dass der Test in keinem einzigen Bericht erscheint. Bei der Option "In diese Berichte einschließen:" können Sie bestimmte Berichte auswählen, in denen der Test erscheinen soll (wobei er in anderen Tests als hier festgelegt nicht erscheint). Bei "Aus diesen Berichten ausschließen:" können Sie festlegen, dass der Test nur in den hier festgelegten Berichten nicht erscheinen soll.

Wenn Sie eine der beiden ersten Optionen im Pop-up-Menü wählen, wird die Liste der Berichte darunter versteckt, aber das ist unbedeutend. Wenn Sie eine der letzten beiden Optionen wählen, wird die Liste gezeigt, standardmäßig mit einer Zeile. Dann können Sie einen Bericht zum Ein-/Ausschließen wählen. Wenn Sie weitere Berichte festlegen wollen, klicken Sie den -Button an, um eine weitere Zeile hinzuzufügen.

  • Tipp: Wenn Sie den Mauszeiger über das Berichte-Pop-up-Menü oder seine Menüeinträge halten, werden Hinweise zu dem entsprechenden Bericht eingeblendet, falls vorhanden (die Standardberichte haben allesamt erklärende Hinweise).
Neuer Bericht:
Klicken Sie diesen Button an, um das Bericht-Infobereich zu öffnen, wo Sie neue Berichte hinzufügen können.

Klicken Sie diesen Button an, um die entsprechende Zeile zu entfernen.

Klicken Sie diesen Button an, um eine neue Zeile hinzuzufügen, damit Sie einen anderen Bericht festlegen können.
