Lizenz-Optionen

Simon ist ein Shareware-Produkt. Sie sind herzlich eingeladen und aufgefordert, Simon erst eine Weile auszuprobieren, bevor Sie es kaufen, um sicherzustellen, dass es Ihre Ansprüche erfüllt. Das ist für Sie vorteilhaft, da Sie so nicht unnötig Geld ausgeben müssen für Software, die vielleicht nicht macht, was Sie wollen. Und für uns ist es vorteilhaft, weil wir so teure Rückbuchungen vermeiden. (Sie würden natürlich Ihr Geld zurückbekommen, aber wir würden auf den Gebühren sitzenbleiben.)

Simon hat eine Testzeit von 14 Benutzungstagen. Das bedeutet, dass Sie Simons vollen Funktionsumfang für 14 Tage gratis benutzen können, wobei nur die Tage gezählt werden, an denen Sie Simon gestartet haben. Wenn Sie Simon eine Woche nicht nutzen, verringert sich die Testzeit nicht. Sie können in diesem Fenster die Anzahl der Tage sehen, die schon vergangen ist. Während der Testzeit sind alle Simon-Features verfügbar, so als ob Sie schon eine Lizenz gekauft hätten; Sie sehen nur hin und wieder dieses Fenster (nicht öfter als einmal pro Tag) als Erinnerung daran, dass Sie noch keine Lizenz gekauft haben. Sobald die 14 Nutzungstage verbraucht sind, werden allerdings einige Funktionen abgeschaltet. Ihre Daten werden weiterhin sichtbar sein, aber Sie können dann, neben anderen Einschränkungen, z.B. keine weiteren Einträge mehr hinzufügen oder Einträge ändern. Stellen Sie daher sicher, dass Sie eine Lizenz kaufen, bevor die Testzeit abläuft, damit Sie problemlos weiterarbeiten können.

Diese Seite des Simon-Assistent-Fensters bietet eine Anzahl Möglichkeiten. Sie können wählen, ob Sie Simon gratis weitertesten wollen. Oder falls Sie schon eine Lizenz haben (die Sie z.B. online in der Simon-Bestellseite ) gekauft haben), können Sie das hier einstellen und dadurch den Lizenz-Editor aufrufen. Oder, falls Sie eine Lizenz kaufen möchten, können Sie eine der Kauf-Optionen wählen, die dann durch die eingebaute Kagi-Kaufhilfe ausgeführt wird.

Das Fenster enthält außerdem Buttons, um Ihren Webbrowser aufzurufen. Der [Website]-Button führt zur Simon-Homepage , während der [Online kaufen]-Button Sie zur Simon-Bestellseite führt, falls Sie lieber im Web oder via TrialPay oder PayPal kaufen wollen, anstatt mit Kagi.

Wenn Sie eine der Kauf-Optionen wählen, erscheint das Kagi-Kaufhilfe-Fenster, nachdem der Kagi-Server kontaktiert wurde. Die Kagi-Kaufhilfe bietet sicheren Einkauf einer Simon-Lizenz, ohne das Programm verlassen zu müssen, und Ihre Seriennummer wird automatisch übertragen. Natürlich bekommen Sie zusätzlich auch eine E-Mail oder einen Brief mit Ihren Lizenz-Informationen, aber Sie brauchen dies beim Kauf über die Kagi-Kaufhilfe nicht wirklich. Wenn der Kauf abgeschlossen ist, öffnet sich der Lizenz-Editor, wo Sie nun Ihren Lizenz-Eintrag sehen können.
