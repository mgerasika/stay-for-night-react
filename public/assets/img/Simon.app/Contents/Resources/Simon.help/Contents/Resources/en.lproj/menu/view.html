View menu

The View menu contains commands to get around the Monitor window.

Sort By: This submenu enables you to sort the current list on various values, and change the sort order.

Absolute Dates / Relative Times: These choices indicate how dates are displayed in the lists: as absolute dates like "2014-12-25 12:37:53" (in your current date format) or relative times like "3 hrs, 25 mins ago".

Show Last Change & Failure: If this option is checked, the Tests list includes two extra lines, showing the most recent change and failure for each test. Uncheck it for a more compact list.

The remaining menu items enable you to change the displayed panes in the window: the lists, showing or hiding the Info pane, and the logs or preview.
