Melder-InfobereichMelder-ArtE-Mail-Melder-ModulE-Mail-Transport-Optionen

Der Dialog "E-Mail-Transport-Optionen" für das E-Mail-Melder-Modul wird gezeigt, wenn Sie den [Optionen]-Button anklicken oder "Server hinzufügen" im Transport-Pop-up-Menü der E-Mail-Melder-Modul wählen.

Transportmethode:
Wählen Sie aus diesem Pop-up-Menü die Methode, mit der Sie E-Mail-Nachrichten versenden wollen. Die Optionen sind SMTP und Mailer, die entsprechend andere Felder in diesem Dialog nutzen. Die meisten Benutzer wählen SMTP.
Mail-Ausgangs-Server:
Geben Sie den vollständigen Domänen-Namen Ihres SMTP-Mailservers an. Wenn Sie unsicher sind, schauen Sie in den Einstellungen Ihres Mail-Programms nach oder fragen Sie Ihren ISP oder Internet-Verwalter.
Notiz: Wenn Sie .Mac benutzen, geben Sie nur "smtp.mac.com" (ohne Anführungszeichen) ein, nicht das Komma und den Benutzernamen, wie es sonst üblich ist.
Server-Port:
Geben Sie die Portnummer ein, die der SMTP-Mailserver benutzt. In den meiste Fällen können Sie den Standardwert 25 benutzen.
SSL (Secure Sockets Layer) verwenden:
Haken Sie diese Option an, wenn Ihr Mailserver SSL-Kommunikation unterstützt, damit Sie eine sichere Verbindung verwenden können.
Authentifizierung:
Wählen Sie die passende Authentifizierungsmethode, die von Ihrem Mailserver unterstützt wird. Wenn Sie eine ausprobieren und sie nicht funktioniert, testen Sie die anderen Optionen; verschiedene Mail-Programme benennen diese Methoden unterschiedlich. Auch hier gilt: Fragen Sie Ihren Internet-Verwalter, falls Sie Hilfe brauchen.
Benutzername:
Geben Sie den Benutzernamen ein, mit dem Sie sich bei Ihrem Mailserver einloggen, wenn Sie irgendeine Authentifizierungsmethode benutzen.
Passwort:
Geben Sie das Passwort für Ihren Mailserver ein, wenn Sie Authentifizierung benutzen. Ihre Eingaben werden nicht gezeigt. Sie können das Feld nicht leer lassen, wenn es benötigt wird, weil Simon Sie nicht während des Versendens danach fragen wird.
Mailer:
Geben Sie den vollen Pfad zu Ihrem Sendmail-Werkzeug ein, zusammen mit allen benötigten Optionen. Diese Methode können Benutzer verwenden, die auf ihrem Computer ein Sendmail-Werkzeug konfiguriert haben. Informationen darüber, wie Sie Sendmail aktivieren, finden Sie auf dieser externen Website.
Löschen:
Klicken Sie diesen Button an, um diese Transportmethode aus dem Transport-Pop-up-Menü zu entfernen. Die Löschung wird ohne weitere Bestätigung ausgeführt.
Abbrechen:
Dieser Button schließt diesen Dialog, ohne Ihre Änderungen zu übernehmen.
OK:
Mit einem Klick auf diesen Button sichern Sie Ihre Änderungen und fügen die Transportmethode dem Transport-Pop-up-Menü hinzu.
