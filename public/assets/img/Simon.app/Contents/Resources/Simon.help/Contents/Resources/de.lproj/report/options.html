Bericht-InfobereichBericht-Optionen

In der Seite "Bericht-Optionen" im Bericht-Infobereich können Sie festlegen, wie oft dieser Bericht ausgegeben werden soll, welche Seiten generiert werden sollen, welche Vorlage verwendet werden soll usw., wie nachfolgend beschrieben:

Aktualisiere Bericht alle:
Legen Sie hier fest, wie oft der Bericht ausgegeben werden soll; wählen Sie eine Zeiteinheit und geben Sie den gewünschten Wert dazu ein, z.B. 20 Minuten oder 2 Stunden. Der Bericht wird direkt nach dem Klicken auf [OK] ausgegeben und erneut dann, wenn die hier angegebene Zeitspanne um ist, sofern Simon läuft und der Bericht nicht gerade bearbeitet wird oder pausiert.
Zusammenfassung erstellen:
Wenn diese Option abgehakt ist, wird eine Zusammenfassung-Seite ausgegeben. Sie enthält üblicherweise eine Tabelle wie die Tests-Tabelle im Simon-Monitor-Fenster, mit Links zu Detailseiten für jeden Test.
Detailseiten generieren:
Wenn diese Option abgehakt ist, wird eine Detailseite für jeden enthaltenen Test ausgegeben. Diese Detailseiten enthalten üblicherweise mehr Informationen über jeden Test.
Vorlage verwenden:
Wählen Sie hier, welche Vorlage dieser Bericht verwenden soll. Verschiedene Vorlagen sind schon eingebaut. Im Kapitel Simon-Extras können Sie weitere aktualisierte und von anderen Anwendern bereitgestellte Vorlagen herunterladen. Lesen Sie auch das Kapitel Bericht-Vorlagen für weitere Informationen über Simons Bericht-Vorlagen.
Im Finder zeigen:
Klicken Sie diesen Buton an, um den gewählten Vorlagen-Ordner im Finder zu zeigen, so dass Sie ihn dort ansehen oder bearbeiten können.
Pausierte Tests überspringen:
Wenn Sie diese Option abhaken, werden alle Tests übersprungen, die zum Zeitpunkt, an dem der Bericht generiert wird, pausieren. Nützlich, wenn Sie routinemäßig pausierende Tests haben und nicht möchten, dass diese den Bericht unnötig überladen.
