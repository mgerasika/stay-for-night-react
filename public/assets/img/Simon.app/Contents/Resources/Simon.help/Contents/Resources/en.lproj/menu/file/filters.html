File menu for Filters window

The File menu for the Filters list contains commands to perform actions on filters:

New Filter...: This function displays the Filter Info pane, described elsewhere.

Duplicate Filter...: This function displays the Filter Info pane using values from the selected filter as a starting point.

Pause Filter: This function displays a Pause Filter sheet, similar to the Pause Test one. This is useful if you want to avoid certain filters, without having to remove them from all of the tests.
